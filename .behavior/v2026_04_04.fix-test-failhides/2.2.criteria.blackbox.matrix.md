## pattern enumeration (supplemental)

### failhide patterns (forbidden) — for rule reference

| scope | pattern | why forbidden |
|-------|---------|---------------|
| prod | `try { ... } catch { /* swallow */ }` | hides error, continues as if ok |
| prod | `try { ... } catch { return null }` | masks failure with null |
| prod | `if (error) { /* ignore */ }` | silent discard |
| test | `if (!condition) { expect(true).toBe(true) }` | fake verification |
| test | `if (!hasResource) { return }` | silent skip |
| test | `expect([0, 1, 2]).toContain(exitCode)` | accepts errors as valid |
| test | `expect.any(Object)` | avoids actual verification |
| test | `it('does x', () => {})` | empty body, no verification |

### error class selection (for failloud)

| who fixes | class | exit code | alias |
|-----------|-------|-----------|-------|
| caller | ConstraintError | 2 | — |
| caller | BadRequestError | 2 | alias to ConstraintError |
| server | MalfunctionError | 1 | — |
| server | UnexpectedCodePathError | 1 | alias to MalfunctionError |
