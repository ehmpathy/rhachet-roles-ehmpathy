# evaluation: pretooluse.allow-rhx-skills

## summary (as implemented)

created a PreToolUse hook that auto-approves rhx/rhachet skill commands to bypass Claude Code's bash safety heuristics. the hook:
1. detects rhx command prefixes
2. rejects commands with shell operators outside quotes (security boundary)
3. returns `permissionDecision: allow` for safe rhx commands
