## subcomponent contracts

these contracts are demanded by the wish (lines 209-255).

### get_one_goal_from_input
```
given('get_one_goal_from_input contract')
  then('exposes: (input: { branch, from?, into? }) => { from: feat|main|branch, into: main|prod }')
  then('throws ConstraintError when from=main and into=main')
  then('infers from=feat when on feature branch and --from omitted')
  then('infers from=main when on main branch and --from omitted')
  then('infers into=main when from=feat and --into omitted')
  then('infers into=prod when from=main and --into omitted')
```

### get_all_flags_from_input
```
given('get_all_flags_from_input contract')
  then('exposes: (argv) => { watch: bool, apply: bool, retry: bool, dirty: block|allow }')
  then('--apply implies watch=true')
  then('defaults: watch=false, apply=false, retry=false, dirty=block')
```

### emit_transport_status
```
given('emit_transport_status contract')
  then('exposes: (input: { of: transport, flags: { apply?, retry? } }) => void')
  then('outputs uniform 🌊 release: {title} header')
  then('outputs check status: 👌|🐢|⚓')
  then('outputs automerge status: 🌴 ...')
  then('outputs hints when applicable')
  then('if apply=true and automerge unfound, enables automerge')
  then('if retry=true and checks failed, triggers rerun')
```

### emit_transport_watch
```
given('emit_transport_watch contract')
  then('exposes: (input: { of: transport }) => void')
  then('outputs 🥥 lets watch header')
  then('polls until completion or 15min timeout')
  then('outputs each poll cycle: 💤 N left, Xs in action, Xs watched')
  then('outputs final status: 👌|⚓|timeout')
  then('shows at least 3 poll cycles when inflight')
```

### get_one_transport_status
```
given('get_one_transport_status contract')
  then('exposes: (input: { of: transport }) => unfound|inflight|passed|failed|merged')
  then('returns unfound when no open PR')
  then('returns merged when PR is merged')
  then('returns inflight when checks in progress')
  then('returns passed when all checks passed')
  then('returns failed when any check failed')
```

### get_one_automerge_status
```
given('get_one_automerge_status contract')
  then('exposes: (input: { of: transport }) => unfound|found|added|merged')
  then('returns unfound when automerge not enabled')
  then('returns found when automerge was already enabled')
  then('returns added when automerge was just enabled')
  then('returns merged when already merged')
```

### emit_one_transport_status_exitcode
```
given('emit_one_transport_status_exitcode contract')
  then('exposes: (input: { status }) => never (exits process)')
  then('exits 0 for passed|merged')
  then('exits non-zero for unfound|inflight|failed')
```

## composition boundaries

demanded by the wish (lines 227-255).

### main flow composition
```
given('git.release main flow')
  then('composes get_one_goal_from_input to determine goal')
  then('composes get_all_flags_from_input to determine flags')
  then('if goal.from=feat, processes feature-branch transport')
  then('if goal.into=prod, processes release-branch and release-tag transports')
  then('each transport: emit_transport_status, then emit_transport_watch if flags.watch')
  then('stops early if any transport is not merged after watch')
```

### transport process composition
```
given('transport process')
  then('emit_transport_status outputs current state')
  then('emit_transport_watch polls until terminal state')
  then('get_one_transport_status returns final state')
  then('emit_one_transport_status_exitcode exits if not merged')
```

### transport adapters
```
given('transport type differences')
  then('feature-branch adapter: PR checks, PR automerge')
  then('release-branch adapter: PR checks, PR automerge')
  then('release-tag adapter: workflow runs (no automerge)')
  then('all adapters expose same contract to emit_transport_*')
```
