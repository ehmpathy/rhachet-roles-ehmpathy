## lessons

### lesson 1: actual execution validates instructions

the strongest evidence that instructions are clear is that they work when executed. the review includes actual command output, not just "i believe this would work".

### lesson 2: copy-paste test is binary

either a command can be copy-pasted and run, or it cannot. no placeholders, no "replace this with your value". the playtest passes this test.

### lesson 3: explicit outcomes prevent ambiguity

"exit 0" is unambiguous. "12 files" is unambiguous. "contains 🐢 radical!" is unambiguous. the playtest uses only unambiguous criteria.
