## lessons

### lesson 1: blueprints may prescribe more than needed

blueprint line 55 prescribed "$ROUTE substitution" but templates contain no $ROUTE references. the implementation correctly omits the unnecessary operation. this is valid — blueprints are conservative.

### lesson 2: test isolation mechanisms are acceptable additions

SKIP_ROUTE_BIND is not in the blueprint but is necessary for integration tests. such test isolation mechanisms are acceptable when:
- they don't change production behavior
- they enable tests without full infrastructure
- they are documented

### lesson 3: wish takes precedence over blueprint for omissions

blueprint template omitted `pnpm fix` but wish specifies the full command sequence with fix. implementation correctly adds `pnpm fix` to match wish intent.
