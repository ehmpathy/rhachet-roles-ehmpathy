# playtest: init.declapract.upgrade

## prerequisites

- pnpm available on PATH
- git repository with at least one commit
- a real declapract repo (one that has `declapract.use.yml`)

## sandbox

- all file operations target `.temp/` or the test repo
- never pollute the repo root

---

## happy paths

### 1. invoke the skill in a declapract repo

**action:**
```sh
# navigate to a repo with declapract.use.yml (e.g., this repo)
cd /home/vlad/git/ehmpathy/_worktrees/rhachet-roles-ehmpathy.vlad.declapract-repair

# invoke the skill
npx rhachet run --skill init.declapract.upgrade
```

**expected outcome:**
- exit code 0
- stdout contains `🐢 radical!` header
- stdout contains `🐚 init.declapract.upgrade` tree
- stdout contains list of 12 created files
- stdout contains `🥥 hang ten!` footer
- `.route/v2026_03_15.declapract.upgrade/` directory created (date matches today)

**verification:**
```sh
ls -la .route/v2026_03_15.declapract.upgrade/
```

**expected files:**
- 1.upgrade.invoke.sh (chmod 555)
- 1.upgrade.invoke.stone
- 2.detect.hazards.stone
- 2.detect.hazards.guard
- 3.1.repair.test.defects.stone
- 3.1.repair.test.defects.guard
- 3.2.reflect.test.defects.stone
- 3.2.reflect.test.defects.guard
- 3.3.repair.cicd.defects.stone
- 3.3.repair.cicd.defects.guard
- 3.4.reflect.cicd.defects.stone
- 3.4.reflect.cicd.defects.guard

### 3. verify findsert semantics (idempotent)

**action:**
```sh

npx rhachet run --skill init.declapract.upgrade
```

## edgey paths

### 1. repo without declapract.use.yml

**action:**
```sh
# create temp repo without declapract.use.yml
cd /tmp
mkdir -p test-no-declapract && cd test-no-declapract
git init
git config user.email "test@test.com"
git config user.name "Test"
echo "hello" > readme.md
git add readme.md
git commit -m "init"

# try to invoke skill
npx rhachet run --skill init.declapract.upgrade
echo "exit code: $?"

# cleanup
cd /tmp && rm -rf test-no-declapract
```

**expected outcome:**
- exit code 2
- stdout contains `not a declapract repo`
- no `.route/` directory created

---

### 2. pnpm not on PATH

**action:**
```sh
# create temp repo with declapract.use.yml
cd /tmp
mkdir -p test-no-pnpm && cd test-no-pnpm
git init
git config user.email "test@test.com"
git config user.name "Test"
echo "declared:" > declapract.use.yml
echo "  - package: declapract-typescript-ehmpathy" >> declapract.use.yml
git add declapract.use.yml
git commit -m "init"

# invoke with minimal PATH (no pnpm)
PATH=/usr/bin:/bin npx rhachet run --skill init.declapract.upgrade
echo "exit code: $?"

# cleanup
cd /tmp && rm -rf test-no-pnpm
```

**expected outcome:**
- exit code 2
- stdout contains `pnpm not found`
- no `.route/` directory created

---

## pass/fail criteria

### pass if:

1. ✓ happy path 1: route created with all 12 files
2. ✓ happy path 2: 1.upgrade.invoke.sh has chmod 555
3. ✓ happy path 3: second invocation reuses same route (findsert)
4. ✓ edgey path 1: exit 2 + "not a declapract repo" when yml absent
5. ✓ edgey path 2: exit 2 + "pnpm not found" when pnpm absent
6. ✓ turtle vibes output present (🐢, 🐚, 🥥)

### fail if:

1. ✗ any exit code mismatch
2. ✗ any expected file absent
3. ✗ 1.upgrade.invoke.sh not executable
4. ✗ duplicate routes created on reinvocation
5. ✗ error message lacks clarity or is absent
6. ✗ turtle vibes absent from output
