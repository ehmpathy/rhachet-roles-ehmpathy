# distill: experience reproductions for init.declapract.upgrade

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| invoke skill in repo with declapract.use.yml | `rhx init.declapract.upgrade` | run with --mode apply | route instantiated, route bound | integration |
| invoke skill in repo without declapract.use.yml | `rhx init.declapract.upgrade` | run | fail with "not a declapract repo" | integration |
| invoke skill without pnpm | `rhx init.declapract.upgrade` | run | fail with "pnpm not found" | integration |
| plan mode preview | `rhx init.declapract.upgrade` | run without --mode | show preview, no files created | integration |
| apply mode creates files | `rhx init.declapract.upgrade --mode apply` | run | all stone/guard files created | integration |
| idempotent apply | `rhx init.declapract.upgrade --mode apply` (twice) | run twice | same route, no duplicate files | integration |
| route bind invoked | `rhx init.declapract.upgrade --mode apply` | run | .route/.bind.*.flag created | integration |

## reproduction feasibility

### test utilities available

from internal code research:
- `genTempDir` — create isolated temp directories
- `spawnSync` — execute skill and capture output
- `setupTempGitRepo` — create git repo with files
- `useBeforeAll` — share setup across tests
- `given/when/then` — BDD structure

### setup required

1. create temp git repo with `declapract.use.yml`
2. ensure pnpm is on PATH (or mock absence via PATH manipulation)
3. run skill via spawnSync
4. assert on exit code, stdout, and file creation

### concrete test sketch

```typescript
import { given, when, then, useBeforeAll } from 'test-fns';
import { genTempDir } from '@ehmpathy/as-command';
import { spawnSync } from 'child_process';
import * as fs from 'fs';
import * as path from 'path';

describe('init.declapract.upgrade', () => {
  const skillPath = path.join(__dirname, '../init.declapract.upgrade.sh');

  given('[case1] repo with declapract.use.yml', () => {
    const scene = useBeforeAll(async () => {
      const tempDir = genTempDir();
      spawnSync('git', ['init'], { cwd: tempDir });
      spawnSync('git', ['config', 'user.email', 'test@test.com'], { cwd: tempDir });
      spawnSync('git', ['config', 'user.name', 'Test'], { cwd: tempDir });
      fs.writeFileSync(
        path.join(tempDir, 'declapract.use.yml'),
        'practices:\n  - declapract-typescript-ehmpathy'
      );
      spawnSync('git', ['add', '-A'], { cwd: tempDir });
      spawnSync('git', ['commit', '-m', 'init'], { cwd: tempDir });
      return { tempDir };
    });

    when('[t0] plan mode (default)', () => {
      then('shows preview without file creation', () => {
        const result = spawnSync('bash', [skillPath], {
          cwd: scene.tempDir,
          encode: 'utf-8',
        });
        expect(result.status).toBe(0);
        expect(result.stdout).toContain('preview:');
        expect(result.stdout).toContain('run with --mode apply');

        const routeDir = path.join(scene.tempDir, '.route');
        expect(fs.existsSync(routeDir)).toBe(false);
      });
    });

    when('[t1] apply mode', () => {
      then('creates route directory and files', () => {
        const result = spawnSync('bash', [skillPath, '--mode', 'apply'], {
          cwd: scene.tempDir,
          encode: 'utf-8',
        });
        expect(result.status).toBe(0);

        // check route directory exists
        const routeDirs = fs.readdirSync(path.join(scene.tempDir, '.route'));
        const routeDir = routeDirs.find(d => d.startsWith('v') && d.includes('declapract.upgrade'));
        expect(routeDir).toBeDefined();

        // check stone files exist
        const routePath = path.join(scene.tempDir, '.route', routeDir!);
        expect(fs.existsSync(path.join(routePath, '1.upgrade.invoke.stone'))).toBe(true);
        expect(fs.existsSync(path.join(routePath, '2.detect.hazards.stone'))).toBe(true);
        expect(fs.existsSync(path.join(routePath, '2.detect.hazards.guard'))).toBe(true);
      });
    });
  });

  given('[case2] repo without declapract.use.yml', () => {
    const scene = useBeforeAll(async () => {
      const tempDir = genTempDir();
      spawnSync('git', ['init'], { cwd: tempDir });
      return { tempDir };
    });

    when('[t0] skill is invoked', () => {
      then('fails with clear error', () => {
        const result = spawnSync('bash', [skillPath], {
          cwd: scene.tempDir,
          encode: 'utf-8',
        });
        expect(result.status).toBe(2);
        expect(result.stderr).toContain('not a declapract repo');
      });
    });
  });
});
```

## gaps

### gap 1: route.bind invocation

**blocked:** the skill needs to invoke `npx rhachet run --repo bhrain --skill route.bind` to bind the route.

**required to unblock:**
- bhrain repo must have route.bind skill available
- or we mock/stub the bind operation in tests

**blocker?** no — can test file creation without bind, then add bind test later

### gap 2: pnpm absence test

**blocked:** need to manipulate PATH to hide pnpm binary

**required to unblock:**
- create fake bin directory without pnpm
- prepend to PATH in test env

**blocker?** no — use PATH injection pattern from internal code research

### gap 3: fnm absence test

**blocked:** fnm check happens at 1.upgrade.invoke stone execution, not at skill invocation

**required to unblock:**
- this is tested via stone execution, not skill test
- stone test would mock fnm absence

**blocker?** no — out of scope for init skill tests; belongs to declapract.upgrade invoke tests
