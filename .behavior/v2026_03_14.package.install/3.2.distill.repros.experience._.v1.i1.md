### setup required

| test scenario | setup |
|---------------|-------|
| success install | temp dir with package.json; no prior install; mock or skip network |
| blocked by security | temp dir; mock advisory endpoint to return vulnerabilities |
| blocked by absent flag | temp dir; minimal setup |
| blocked by extant package | temp dir; pre-install target package |
| success upgrade | temp dir; pre-install target package at older version |
| @latest expansion | temp dir; mock npm view or use real registry |
| pnpm vs npm | temp dir with/without package-lock.json |
| reason via @stdin | temp dir; spawn with `input:` option |

### concrete test sketch

```typescript
import { spawnSync } from 'child_process';
import * as fs from 'fs';
import * as path from 'path';
import * as os from 'os';
import { given, then, when, useBeforeAll } from 'test-fns';

const skillPath = path.join(__dirname, 'set.package.install.sh');

const setupTempDir = (): string => {
  const tempDir = fs.mkdtempSync(path.join(os.tmpdir(), 'pkg-install-test-'));
  fs.writeFileSync(
    path.join(tempDir, 'package.json'),
    JSON.stringify({ name: 'test-project', version: '1.0.0', dependencies: {} }, null, 2),
  );
  return tempDir;
};

const runSkillWithStdin = (
  args: string,
  stdin: string,
  cwd: string,
): { stdout: string; stderr: string; exitCode: number } => {
  const result = spawnSync(
    'bash',
    [skillPath, ...args.split(' ').filter(Boolean)],
    {
      cwd,
      encoding: 'utf-8',
      stdio: ['pipe', 'pipe', 'pipe'],
      input: stdin,
    },
  );
  return {
    stdout: result.stdout ?? '',
    stderr: result.stderr ?? '',
    exitCode: result.status ?? 1,
  };
};

describe('set.package.install.sh', () => {
  given('[case1] all flags valid, audit passes', () => {
    const tempDir = useBeforeAll(() => setupTempDir());

    when('[t0] skill invoked with @stdin reason', () => {
      then('exit code is 0', () => {
        const result = runSkillWithStdin(
          '--package test-fns --at 1.0.0 --for prod --reason @stdin',
          'unit test framework, abstracts jest boilerplate',
          tempDir,
        );
        expect(result.exitCode).toBe(0);
      });

      then('reason file created', () => {
        // verify .route/v*.package.install/3.reason.for_test-fns.*.md exists
      });
    });
  });

  given('[case2] --package flag omitted', () => {
    when('[t0] skill invoked without --package', () => {
      then('exit code is 2', () => {
        const result = runSkillWithStdin('--at 1.0.0 --for prod --reason "test"', '', os.tmpdir());
        expect(result.exitCode).toBe(2);
      });

      then('error mentions --package', () => {
        const result = runSkillWithStdin('--at 1.0.0 --for prod --reason "test"', '', os.tmpdir());
        expect(result.stderr).toContain('--package');
      });
    });
  });
});
```

## summary

all experiences can be reproduced via integration tests. key patterns:
- spawnSync for skill invocation
- temp dir for isolation
- stdin input for @stdin reason
- real registry for success paths, mock for failure paths
