### [PATTERN-2] skill runner operation
**source**: `src/domain.roles/mechanic/skills/get.package.docs.integration.test.ts:18-36`
**mark**: [REUSE]

```typescript
const runSkill = (
  args: string,
): { stdout: string; stderr: string; exitCode: number } => {
  const result = spawnSync(
    'bash',
    [skillPath, ...args.split(' ').filter(Boolean)],
    {
      cwd: process.cwd(),
      encoding: 'utf-8',
      stdio: ['pipe', 'pipe', 'pipe'],
    },
  );

  return {
    stdout: result.stdout ?? '',
    stderr: result.stderr ?? '',
    exitCode: result.status ?? 1,
  };
};
```

**relevance**: use same spawnSync pattern for bash skill invocation. will need stdin support for @stdin tests.

const setupTempDir = (): string => {
  const tempDir = fs.mkdtempSync(path.join(os.tmpdir(), 'skill-test-'));
