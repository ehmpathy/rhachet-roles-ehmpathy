## why it holds

1. **critical paths covered** — security block and boundary conditions tested
2. **both skills have edgey paths** — install has 4, upgrade has 1 (not installed)
3. **flag validation pattern shown** — step 10 demonstrates error behavior
4. **gaps are low-medium severity** — core behavioral edges are in playtest

the playtest covers the essential edge cases. gaps left are either impractical to test (network errors) or already covered by unit tests (scoped packages).
