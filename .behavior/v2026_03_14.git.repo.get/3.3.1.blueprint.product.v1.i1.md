# blueprint: git.repo.get

## summary

build a shell skill that enables mechanic to search and browse repos without permission prompts. the skill has three subcommands:
- `repos` вЂ” enumerate repos that match a glob pattern
- `files` вЂ” list files in a repo
- `lines` вЂ” search or read file contents

local-first design with cloud fallback via gh cli. output uses turtle vibes format.
