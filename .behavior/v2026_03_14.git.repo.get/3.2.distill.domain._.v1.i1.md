# distill: domain model

## overview

git.repo.get is a shell skill, not a TypeScript service. domain objects are conceptual and represented via output format and internal variables.

---

## usecases

| usecase | contract |
|---------|----------|
| list repos | `rhx git.repo.get repos --repos 'org/*'` |
| list files | `rhx git.repo.get files --in org/name` |
| search lines | `rhx git.repo.get lines --in org/name --words 'pattern'` |
| read file | `rhx git.repo.get lines --in org/name --paths 'src/file.ts'` |

---

## domain objects

### RepoSlug (literal)

represents a fully qualified repository identifier.

```
@scope/org/repo

scope = local | cloud | all
org = github organization or user
repo = repository name
```

**examples:**
```
@local/ehmpathy/domain-objects
@cloud/ehmpathy/new-library
@all/ehmpathy/*
```

**in shell:**
```bash
SLUG="@local/ehmpathy/domain-objects"
SCOPE="${SLUG%%/*}"        # @local
ORG_REPO="${SLUG#*/}"       # ehmpathy/domain-objects
ORG="${ORG_REPO%%/*}"       # ehmpathy
REPO="${ORG_REPO#*/}"       # domain-objects
```

### RepoSource (literal)

indicates where repo data comes from.

```
local | cloud
```

**purpose:** shown in output so user knows source of results.

### LocalClone (entity-like)

a git repository cloned locally.

**attributes:**
- path: absolute path to clone (e.g., ~/git/ehmpathy/domain-objects)
- org: organization name
- name: repository name
- defaultBranch: main or master (detected)

**enumeration:**
```bash
find ~/git -maxdepth 2 -name ".git" -type d | while read gitdir; do
  path="$(dirname "$gitdir")"
  org="$(basename "$(dirname "$path")")"
  name="$(basename "$path")"
done
```

### CloudRepo (entity-like)

a github repository accessible via gh cli.

**attributes:**
- url: github url
- org: organization or user
- name: repository name
- defaultBranch: detected via gh api

**enumeration:**
```bash
gh repo list ehmpathy --json name,url | jq -r '.[] | "\(.name)\t\(.url)"'
```

### FileEntry (literal)

a file within a repository.

**attributes:**
- path: relative path within repo
- source: local or cloud

**enumeration (local):**
```bash
git ls-tree -r --name-only origin/main
```

**enumeration (cloud):**
```bash
gh api repos/{owner}/{repo}/git/trees/main?recursive=1 --jq '.tree[] | select(.type == "blob") | .path'
```

### LineMatch (literal)

a line that matches a search pattern.

**attributes:**
- file: FileEntry
- lineNumber: integer
- content: the line text
- isMatch: boolean (true for match, false for context)

**search (local):**
```bash
git grep -n "pattern" origin/main -- "*.ts"
```

### LineContext (literal)

a group of lines around a match.

**attributes:**
- file: FileEntry
- matches: LineMatch[]
- contextBefore: number
- contextAfter: number

**format (radius=3):**
```
     1: context line
     2: context line
 >   3: match line
     4: context line
     5: context line
```

## domain operations

### repos subcommand

| operation | purpose |
|-----------|---------|
| getAllLocalClones | enumerate ~/git/ for clones |
| getAllCloudRepos | enumerate via gh repo list |
| filterReposByGlob | match slug against glob pattern |

**composition:**
```
getAllRepos(glob, scope) =
  if scope == local: getAllLocalClones() |> filterReposByGlob(glob)
  if scope == cloud: getAllCloudRepos() |> filterReposByGlob(glob)
  if scope == all: merge(getAllLocalClones(), getAllCloudRepos()) |> filterReposByGlob(glob)
```

### files subcommand

| operation | purpose |
|-----------|---------|
| getOneLocalClone | find local clone by org/name |
| getOneCloudRepo | find cloud repo by org/name |
| getAllFilesAtRef | list files via git ls-tree |
| filterFilesByGlob | match paths against glob |
| filterFilesByContent | match files that contain pattern |

**composition:**
```
getAllFiles(repo, paths, words, ref) =
  source = getOneLocalClone(repo) ?? getOneCloudRepo(repo)
  files = getAllFilesAtRef(source, ref)
  if paths: files = filterFilesByGlob(files, paths)
  if words: files = filterFilesByContent(files, words)
  return files
```

### lines subcommand (search mode)

| operation | purpose |
|-----------|---------|
| searchLinesInRepo | git grep at ref |
| getContextAroundMatch | expand match with radius lines |

**composition:**
```
searchLines(repo, words, paths, radius, ref) =
  source = getOneLocalClone(repo) ?? getOneCloudRepo(repo)
  matches = searchLinesInRepo(source, words, paths, ref)
  return matches.map(m => getContextAroundMatch(m, radius))
```

### lines subcommand (read mode)

| operation | purpose |
|-----------|---------|
| getFileContent | git show ref:path |

**composition:**
```
readFile(repo, path, ref) =
  source = getOneLocalClone(repo) ?? getOneCloudRepo(repo)
  return getFileContent(source, path, ref)
```

## relationships

```
RepoSlug
  └─ parsed into (scope, org, repo)
       └─ resolves to LocalClone | CloudRepo
            └─ contains FileEntry[]
                 └─ contains LineMatch[] (via search)
```

### source resolution

```
@all scope:
  1. try getOneLocalClone(org, repo)
  2. if not found, try getOneCloudRepo(org, repo)
  3. output shows (local) or (cloud)

@local scope:
  1. getOneLocalClone(org, repo)
  2. if not found, error (no fallback)

@cloud scope:
  1. getOneCloudRepo(org, repo)
  2. local clones ignored
```

## composition for wish fulfillment

### "stop permission asks"

### "stop fails to find stuff"

local-first with cloud fallback → reliable results

### "more reliable search"

### "clarity on accessible repos"

`repos --repos 'org/*'` shows what's available, marked (local) or (cloud)

## summary

| concept | representation |
|---------|----------------|
| RepoSlug | `@scope/org/repo` string |
| LocalClone | directory at `~/git/org/repo/` |
| CloudRepo | github url via gh cli |
| FileEntry | path within repo |
| LineMatch | line number + content + isMatch flag |
| LineContext | group of lines with radius around match |

| operation | implementation |
|-----------|----------------|
| enumerate local | `find ~/git -name .git` |
| enumerate cloud | `gh repo list org` |
| list files | `git ls-tree -r --name-only ref` |
| search lines | `git grep -n pattern ref` |
| read file | `git show ref:path` |
