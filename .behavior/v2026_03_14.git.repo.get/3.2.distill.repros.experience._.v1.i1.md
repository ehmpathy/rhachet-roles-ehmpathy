# distill: experience reproductions

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| list repos | `rhx git.repo.get repos` | `--repos 'ehmpathy/*'` | shows ehmpathy repos with (local)/(cloud) | integration |
| list files | `rhx git.repo.get files` | `--repo ehmpathy/domain-objects` | shows file tree | integration |

| no prompts | `rhx git.repo.get *` | any command | no permission prompt appears | permission check |

## reproduction feasibility

### repos enumeration

**test utilities available:**
- genTempDir with git init
- spawnSync for skill invocation

**setup required:**
- create temp ~/git/ structure with multiple orgs and repos
- override GIT_REPO_ROOT env var to point to temp structure

**test sketch:**
```typescript
given('[case1] local repos in temp git root', () => {
  const gitRoot = setupLocalRepos([
    { org: 'ehmpathy', name: 'domain-objects', files: { 'src/index.ts': '...' } },
    { org: 'ehmpathy', name: 'sql-code-generator', files: { 'src/index.ts': '...' } },
    { org: 'ahbode', name: 'protools-api', files: { 'src/index.ts': '...' } },
  ]);

  when('[t0] repos --repos ehmpathy/*', () => {
    then('shows ehmpathy repos', () => {
      const result = runRepoGetCommand({
        subcommand: 'repos',
        repos: 'ehmpathy/*',
        gitRoot,
      });
      expect(result.stdout).toContain('domain-objects');
      expect(result.stdout).toContain('sql-code-generator');
      expect(result.stdout).not.toContain('protools-api');
    });
  });
});
```

### search lines

**test utilities available:**
- genTempDir with git init
- spawnSync for skill invocation

**setup required:**
- create temp repo with known content
- content must include searchable patterns

**test sketch:**
```typescript
given('[case1] repo with searchable content', () => {
  const gitRoot = setupLocalRepos([
    {
      org: 'test-org',
      name: 'test-repo',
      files: {
        'src/index.ts': 'export class DomainEntity {\n  // entity code\n}',
      },
    },
  ]);

  when('[t0] lines --repo test-org/test-repo --words DomainEntity', () => {
    then('shows match with context', () => {
      const result = runRepoGetCommand({
        subcommand: 'lines',
        repo: 'test-org/test-repo',
        words: 'DomainEntity',
        gitRoot,
      });
      expect(result.stdout).toContain('DomainEntity');
      expect(result.stdout).toContain('src/index.ts');
      expect(result.stdout).toContain('>'); // match marker
    });
  });
});
```

### cloud fallback

**test utilities available:**
- mock gh cli via PATH injection (see howto.mock-cli-via-path brief)

**setup required:**
- no local repo at the slug
- mock gh cli to return expected data

**test sketch:**
```typescript
given('[case1] repo not cloned locally', () => {
  const gitRoot = setupLocalRepos([]); // no repos
  const fakeBinDir = setupMockGhCli({
    'repo view test-org/cloud-repo': { defaultBranch: 'main' },
    'api repos/test-org/cloud-repo/...': { files: [...] },
  });

  when('[t0] files --repo test-org/cloud-repo', () => {
    then('falls back to cloud', () => {
      const result = runRepoGetCommand({
        subcommand: 'files',
        repo: 'test-org/cloud-repo',
        gitRoot,
        env: { PATH: `${fakeBinDir}:${process.env.PATH}` },
      });
      expect(result.stdout).toContain('(cloud)');
    });
  });
});
```

### no permission prompts

**test utilities available:**
- read claude permission settings
- verify allowlist entries

**setup required:**
- check init.claude.permissions.jsonc includes rhx git.repo.get

**test sketch:**
```typescript
given('[case1] permission settings', () => {
  when('[t0] permissions are checked', () => {
    then('git.repo.get is pre-approved', () => {
      const permissions = JSON.parse(
        fs.readFileSync('src/domain.roles/mechanic/inits/init.claude.permissions.jsonc', 'utf-8')
      );
      const allowlist = permissions.permissions.allow;
      const hasEntry = allowlist.some(
        (entry: string) => entry.includes('git.repo.get')
      );
      expect(hasEntry).toBe(true);
    });
  });
});
```

## gaps

### cloud operations require mock

**what is blocked:**
- cloud operations need gh cli which requires auth
- cannot run real gh commands in CI without tokens

**what is required to unblock:**
- use PATH mock pattern (documented in briefs)
- or skip cloud tests in CI, run locally with auth

**is this a blocker?**
- no — local-first tests cover core behavior
- cloud tests can use mock pattern
- defer real cloud tests to manual verification

### multi-repo search

**what is blocked:**
- `lines --repos 'org/*'` requires iteration over multiple repos
- test setup more complex

**what is required to unblock:**
- extend setupLocalRepos to create multiple repos
- test verifies results aggregated from all repos that match

**is this a blocker?**
- no — can defer to later phase
- single-repo search covers core behavior
