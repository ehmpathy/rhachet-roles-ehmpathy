## pattern.1 = genTempDir for isolated test directories [REUSE]

### source

`test-fns` package via imports in `*.integration.test.ts` files

### the pattern

```typescript
import { genTempDir } from 'test-fns';

const tempDir = genTempDir({
  slug: 'git-repo-get-test',
  git: true,
  symlink: [{ at: 'node_modules', to: 'node_modules' }],
});
```

### key features

- creates timestamped temp directory in `.temp/`
- `git: true` initializes git repo with initial commit
- `symlink` links repo artifacts for skill access
- auto-prunes directories older than 7 days
- slug helps identify which test created the directory

### application

git.repo.get tests will use `genTempDir({ git: true })` to create test repos with controlled state.

### key conventions

- `[caseN]` labels for given blocks
- `[tN]` labels for when blocks
- each then block = one behavioral assertion
- case numbers reset per describe

## pattern.4 = runInTempGitRepo wrapper [REUSE]

### source

`src/domain.roles/mechanic/skills/git.commit/git.commit.set.integration.test.ts`

### the pattern

```typescript
const runInTempGitRepo = (args: {
  files?: Record<string, string>;
  meterState?: { uses: number; push: string };
  commitArgs: string[];
  stdin?: string;
}): { stdout: string; stderr: string; exitCode: number; tempDir: string } => {
  const tempDir = genTempDir({ slug: 'git-commit-set-test', git: true, ... });

  // setup state
  if (args.files) {
    for (const [filePath, content] of Object.entries(args.files)) {
      fs.writeFileSync(path.join(tempDir, filePath), content);
    }
  }

  // run skill
  const result = spawnSync('bash', [scriptPath, ...args.commitArgs], {
    cwd: tempDir,
    encoding: 'utf-8',
    stdio: ['pipe', 'pipe', 'pipe'],
  });

  return { stdout, stderr, exitCode, tempDir };
};
```

## extension needed: local repo fixture setup

git.repo.get needs a setup pattern for local repos:

```typescript
const setupLocalRepos = (repos: {
  org: string;
  name: string;
  files: Record<string, string>;
}[]): { gitRoot: string; cleanup: () => void } => {
  const gitRoot = path.join(genTempDir({ slug: 'git-root' }), 'git');
  fs.mkdirSync(gitRoot, { recursive: true });

  for (const repo of repos) {
    const repoPath = path.join(gitRoot, repo.org, repo.name);
    fs.mkdirSync(repoPath, { recursive: true });

    // init git
    execSync('git init', { cwd: repoPath });
    execSync('git checkout -b main', { cwd: repoPath });

    // create files
    for (const [filePath, content] of Object.entries(repo.files)) {
      const fullPath = path.join(repoPath, filePath);
      fs.mkdirSync(path.dirname(fullPath), { recursive: true });
      fs.writeFileSync(fullPath, content);
    }

    // commit
    execSync('git add .', { cwd: repoPath });
    execSync('git commit -m "initial"', { cwd: repoPath });
  }

  return {
    gitRoot,
    cleanup: () => fs.rmSync(gitRoot, { recursive: true, force: true }),
  };
};
```
