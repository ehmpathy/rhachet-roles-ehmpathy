### get_git_root (line 17-19)

```bash
get_git_root() {
  echo "${GIT_REPO_ROOT:-$HOME/git}"
}
```

**why unique**: returns the root directory where local clones live (`~/git`). no other skill needs this — git.commit and git.rebase operate on the current working directory, not external repos.

### get_files_at_ref (lines 154-160) and get_files_at_ref_cloud (lines 167-173)

```bash
get_files_at_ref() {
  git ls-tree -r --name-only "$ref" 2>/dev/null
}

get_files_at_ref_cloud() {
  gh api "repos/$slug/git/trees/$ref?recursive=1" \
    --jq '.tree[] | select(.type == "blob") | .path'
}
```

**why unique**: lists files in external repos at specific refs. git.commit/git.rebase never need file enumeration — they work with staged changes or conflict files.

### search_lines (lines 180-187) and search_lines_cloud (lines 195-202)

```bash

search_lines_cloud() {
  gh search code "$pattern" --repo "$slug" --json path --jq '.[].path'

```

**why unique**: content search across repo files. no other skill searches file contents — git.commit validates message format, git.rebase handles conflicts.

### read_file (lines 209-216) and read_file_cloud (lines 223-231)

```bash
read_file() {
  git show "$ref:$file_path" 2>/dev/null
}

read_file_cloud() {
  gh api "repos/$slug/contents/$file_path?ref=$ref" \
    --jq '.content' | base64 -d
}
```

**why unique**: reads file content from external repos. git.commit/git.rebase never read arbitrary files — they work with worktree state.

### get_default_branch (lines 134-147)

```bash
get_default_branch() {
  if [[ "$is_local" == "local" ]]; then
    git symbolic-ref refs/remotes/origin/HEAD | sed 's@^refs/remotes/origin/@@'
  else
    gh repo view "$slug" --json defaultBranchRef --jq '.defaultBranchRef.name'
  fi
}
```

**why potentially shared but not**: git.commit.operations.sh has `get_closest_ancestor_branch()` which detects the base branch for the current repo. however, git.repo.get's `get_default_branch()` operates on external repos (local clone path or github slug). the contexts differ enough that sharing would be forced.
