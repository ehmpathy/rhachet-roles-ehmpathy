### issue found and fixed

**issue**: guardBorder case7 failed

**root cause**: posttooluse.guardBorder.onWebfetch.sh unconditionally sourced ~/.config/rhachet/apikeys.env, which overrode the test-provided empty XAI_API_KEY=''.

**fix applied**: changed the condition to check if XAI_API_KEY is already set before source:

```bash
# before
if [[ -f ~/.config/rhachet/apikeys.env ]]; then
  source ~/.config/rhachet/apikeys.env
fi

# after
if [[ -z "${XAI_API_KEY+x}" && -f ~/.config/rhachet/apikeys.env ]]; then
  source ~/.config/rhachet/apikeys.env
fi
```

**verification**: re-ran full test suite after fix — all 104 tests passed.
