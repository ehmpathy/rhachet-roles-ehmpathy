## pattern.2 = shared operations (git.branch.rebase.operations.sh)

**relation to wish:** [REUSE] — `is_rebase_in_progress()` for guard, `get_conflict_files()` for detection

**what it does:**
- single source of truth for rebase state detection
- detects rebase via git internals (`.git/rebase-merge` or `.git/rebase-apply`)
- gets conflict files via `git diff --name-only --diff-filter=U`
- works in both regular repos and worktrees

**citations:**

[3] rebase detection function:
```bash
is_rebase_in_progress() {
  local git_dir
  git_dir=$(get_git_dir)
  if [[ -d "$git_dir/rebase-merge" ]] || [[ -d "$git_dir/rebase-apply" ]]; then
    return 0  # true
  fi
  return 1  # false
}
```
— git.branch.rebase.operations.sh:26-33

[4] conflict files function:
