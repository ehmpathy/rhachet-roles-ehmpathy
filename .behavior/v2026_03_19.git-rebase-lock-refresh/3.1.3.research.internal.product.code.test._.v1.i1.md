## pattern.3 = runSkill helper

**relation to wish:** [REUSE] — same helper for lock refresh tests

**what it does:**
- executes skill via spawnSync bash
- captures stdout, stderr, status
- passes env with PATH

**citations:**

[4] runSkill helper:
```typescript
const runSkill = (
  tempDir: string,
  args: string[] = [],
): { stdout: string; stderr: string; status: number | null } => {
  const result = spawnSync('bash', [SKILL_PATH, ...args], {
    cwd: tempDir,
    encoding: 'utf-8',
    env: {
      ...process.env,
      PATH: process.env.PATH,
    },
  });

  return {
    stdout: result.stdout || '',
    stderr: result.stderr || '',
    status: result.status,
  };
};
```

**relation to wish:** [REUSE] — for error case tests without full conflict setup

**what it does:**
- creates `.git/rebase-merge` dir with minimal files
- triggers rebase detection without actual conflicts
- faster for guard/error case tests

**citations:**

[5] simulated rebase setup:
```typescript
const setupSimulatedRebase = (): string => {
  // ...git init...

  // simulate rebase in progress (without real conflicts)
  const rebaseMergeDir = path.join(tempDir, '.git', 'rebase-merge');
  fs.mkdirSync(rebaseMergeDir, { recursive: true });
  fs.writeFileSync(
    path.join(rebaseMergeDir, 'head-name'),
    'refs/heads/feature/test-branch',
  );
  fs.writeFileSync(path.join(rebaseMergeDir, 'onto'), 'abc123def456');
  fs.writeFileSync(path.join(rebaseMergeDir, 'msgnum'), '1');
  fs.writeFileSync(path.join(rebaseMergeDir, 'end'), '2');

};
```
