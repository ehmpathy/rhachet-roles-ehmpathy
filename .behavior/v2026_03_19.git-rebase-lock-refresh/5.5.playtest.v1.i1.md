# playtest: git.branch.rebase lock refresh

## prerequisites

- a local repo with pnpm installed
- git configured with user.name and user.email
- rhachet built and linked (`npm run build && npx rhachet roles link`)

## sandbox

all operations target a temporary directory in `.temp/` ‚Äî no pollution of repo root.

---

## happy paths

### 1. take lock file shows suggestion

**setup:**
```sh
cd .temp
mkdir playtest-rebase && cd playtest-rebase
git init
echo '{"name": "test", "version": "1.0.0"}' > package.json
pnpm install
git add -A && git commit -m "initial"
git checkout -b feature
echo '{"name": "test", "version": "1.0.0", "description": "added"}' > package.json
pnpm install
git add -A && git commit -m "feature change"
git checkout main
echo '{"name": "test", "version": "1.0.0", "author": "main"}' > package.json
pnpm install
git add -A && git commit -m "main change"
git checkout feature
git rebase main
```

**action:**
```sh
rhx git.branch.rebase take --whos theirs pnpm-lock.yaml
```

**expected outcome:**
- output shows `üê¢ righteous!`
- output shows `‚îú‚îÄ settled` with `pnpm-lock.yaml ‚úì`
- output shows `‚îú‚îÄ lock taken, refresh it with: ‚ö°`
- output shows `‚îÇ  ‚îî‚îÄ rhx git.branch.rebase lock refresh`

**pass:** suggestion appears in output
**fail:** no suggestion or error

---

### 2. lock refresh regenerates and stages

**action** (continue from step 1):
```sh
rhx git.branch.rebase lock refresh
```

**expected outcome:**
- output shows `üê¢ shell yeah!`
- output shows `‚îú‚îÄ detected: pnpm`
- output shows `‚îú‚îÄ run: pnpm install`
- output shows `‚îú‚îÄ staged` with `pnpm-lock.yaml ‚úì`
- `git status` shows `pnpm-lock.yaml` staged (green)

**pass:** lock is regenerated and staged
**fail:** lock not staged or install error

---

### 3. continue rebase succeeds

**action** (continue from step 2):
```sh
rhx git.branch.rebase continue
```

**expected outcome:**
- rebase completes without conflict
- `git log --oneline` shows both commits

**pass:** rebase completes
**fail:** rebase fails or conflicts remain

## edgey paths

**action:**
```sh

rhx git.branch.rebase lock refresh
```
