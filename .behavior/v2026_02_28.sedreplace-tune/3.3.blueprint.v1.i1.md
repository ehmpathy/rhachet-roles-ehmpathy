# blueprint: sedreplace safety tune

## overview

implement two safety guards for sedreplace:
1. **hardblock unplanned applies** — track plans in `.claude/sedreplace.nudges.local.json`, block `--mode apply` without matched plan
2. **hardblock unbound globs** — reject any `--glob` that starts with `**/`

plus:
- update all docs and permissions to use single-quoted globs `'src/**/*.ts'`
- verify gitignore exclusion works (already implemented via `--exclude-standard`, add test)

## codepaths

### sedreplace.sh

```
sedreplace.sh
├─ [○] parse arguments (--old, --new, --glob, --mode)
├─ [○] validate required args
├─ [+] validate glob pattern
│   ├─ if --glob starts with "**/" → block with helpful message
│   └─ else → continue
├─ [+] plan tracker (HARDNUDGE pattern)
│   ├─ compute plan key = sha256(old + new + glob)
│   ├─ if mode == "plan"
│   │   └─ record plan key in .claude/sedreplace.nudges.local.json
│   ├─ if mode == "apply"
│   │   ├─ check if plan key exists in nudge file
│   │   ├─ if not found → block with "must plan first" message
│   │   └─ if found → continue (allow apply)
│   └─ cleanup stale entries (>1 hour old)
├─ [○] find files that match glob
├─ [○] grep for pattern
├─ [○] plan mode: show diff
└─ [○] apply mode: execute sed
```

### plan tracker state

```json
// .claude/sedreplace.nudges.local.json
{
  "abc123...": {                    // sha256(old + new + glob)
    "time": 1772389362,             // unix timestamp
    "old": "getUserById",           // pattern
    "new": "findUserByUuid",        // replacement
    "glob": "src/**/*.ts"           // glob (or null)
  }
}
```

## contracts

### glob validation

**input:** `--glob` value (string or empty)

**logic:**
```bash
if [[ -n "$GLOB_FILTER" && "$GLOB_FILTER" == \*\*/* ]]; then
  # block: unbound glob
  exit 2
fi
```

**output (blocked):**
```
🐢 bummer dude...

🐚 sedreplace
   └─ ✋ blocked: unbound glob

glob patterns that start with **/ are too broad and can cause unexpected changes.

use a bounded glob instead:
  --glob 'src/**/*.ts'     (files in src/)
  --glob 'src/**/*.test.ts' (test files in src/)
  --glob '*.ts'            (files in root only)
```

### plan tracker

**input:** `--old`, `--new`, `--glob`, `--mode`

**logic:**
```bash
# compute plan key
PLAN_KEY=$(echo -n "${OLD_PATTERN}|${NEW_PATTERN}|${GLOB_FILTER}" | sha256sum | cut -d' ' -f1)

if [[ "$MODE" == "plan" ]]; then
  # record plan
  record_plan "$PLAN_KEY" "$OLD_PATTERN" "$NEW_PATTERN" "$GLOB_FILTER"
fi

if [[ "$MODE" == "apply" ]]; then
  # check for plan
  if ! plan_exists "$PLAN_KEY"; then
    # block
    exit 2
  fi
fi
```

**output (blocked):**
```
🐢 bummer dude...

🐚 sedreplace
   └─ ✋ blocked: cannot apply without plan

to prevent unexpected consequences, sedreplace requires you to preview changes before apply.

run without --mode (or with --mode plan) first, then re-run with --mode apply.
```

## tests

### integration tests (sedreplace.integration.test.ts)

```
describe('sedreplace', () => {
  given('[case1] unbound glob', () => {
    when('[t0] --glob "**/*.ts" is used', () => {
      then('it blocks with helpful message', async () => {
        const result = await runSedreplace({
          old: 'foo',
          new: 'bar',
          glob: '**/*.ts'
        });
        expect(result.exitCode).toEqual(2);
        expect(result.stderr).toContain('unbound glob');
      });
    });
  });

  given('[case2] bounded glob', () => {
    when('[t0] --glob "src/**/*.ts" is used', () => {
      then('it allows the operation', async () => {
        const result = await runSedreplace({
          old: 'foo',
          new: 'bar',
          glob: 'src/**/*.ts'
        });
        expect(result.exitCode).toEqual(0);
      });
    });
  });

  given('[case3] apply without plan', () => {
    when('[t0] --mode apply is used without prior plan', () => {
      then('it blocks with helpful message', async () => {
        // ensure no prior plan
        await clearNudgeFile();

        const result = await runSedreplace({
          old: 'foo',
          new: 'bar',
          mode: 'apply'
        });
        expect(result.exitCode).toEqual(2);
        expect(result.stderr).toContain('cannot apply without plan');
      });
    });
  });

  given('[case4] apply after plan', () => {
    when('[t0] plan is run first', () => {
      then('apply succeeds and changes files', async () => {
        // setup: create file with pattern
        await createTempFile('src/test-apply.ts', 'const foo = 1;');

        // run plan first
        await runSedreplace({ old: 'foo', new: 'bar', glob: 'src/**/*.ts' });

        // now apply
        const result = await runSedreplace({
          old: 'foo',
          new: 'bar',
          glob: 'src/**/*.ts',
          mode: 'apply'
        });
        expect(result.exitCode).toEqual(0);

        // verify file was changed
        const content = await readFile('src/test-apply.ts');
        expect(content).toContain('const bar = 1;');
      });
    });
  });

  given('[case5] apply with different glob than plan', () => {
    when('[t0] plan is run with one glob, apply with another', () => {
      then('it blocks (plan mismatch)', async () => {
        // plan with src/
        await runSedreplace({ old: 'foo', new: 'bar', glob: 'src/**/*.ts' });

        // apply with different glob (test files)
        const result = await runSedreplace({
          old: 'foo',
          new: 'bar',
          glob: 'src/**/*.test.ts',
          mode: 'apply'
        });
        expect(result.exitCode).toEqual(2);
        expect(result.stderr).toContain('cannot apply without plan');
      });
    });
  });

  given('[case6] gitignored files', () => {
    when('[t0] pattern exists in both src/ and .gitignored dir', () => {
      then('only non-gitignored files are included', async () => {
        // setup: create gitignored dir and files
        await ensureDir('.test-ignored');
        await appendToGitignore('.test-ignored/');
        await createTempFile('src/test-gitignore.ts', 'const testGitignoreVar = 1;');
        await createTempFile('.test-ignored/file.ts', 'const testGitignoreVar = 1;');

        // run WITHOUT glob - should search all non-gitignored files
        const result = await runSedreplace({
          old: 'testGitignoreVar',
          new: 'renamedVar'
          // no glob = all files, gitignore should exclude .test-ignored/
        });

        // verify: src/ file included, .test-ignored/ excluded
        expect(result.stdout).toContain('src/test-gitignore.ts');
        expect(result.stdout).not.toContain('.test-ignored');
      });
    });
  });

  given('[case7] stale plan expiry', () => {
    when('[t0] plan was recorded more than 1 hour ago', () => {
      then('apply is blocked (stale plan removed)', async () => {
        // setup: inject stale plan into nudge file
        const staleTime = Date.now() - (2 * 60 * 60 * 1000); // 2 hours ago
        await injectStaleNudge({
          old: 'stalePattern',
          new: 'staleReplacement',
          glob: 'src/**/*.ts',
          time: staleTime
        });

        // apply should fail because stale entries are cleaned up
        const result = await runSedreplace({
          old: 'stalePattern',
          new: 'staleReplacement',
          glob: 'src/**/*.ts',
          mode: 'apply'
        });
        expect(result.exitCode).toEqual(2);
        expect(result.stderr).toContain('cannot apply without plan');
      });
    });
  });
});

## unit tests (sedreplace.test.ts)

```
describe('isUnboundGlob', () => {
  given('[case1] patterns that start with **/', () => {
    const cases = [
      { glob: '**/*.ts', expected: true },
      { glob: '**/foo', expected: true },
      { glob: '**/*', expected: true },
    ];
    cases.forEach(({ glob, expected }) => {
      then(`${glob} → ${expected}`, () => {
        expect(isUnboundGlob(glob)).toEqual(expected);
      });
    });
  });

  given('[case2] patterns that do not start with **/', () => {
    const cases = [
      { glob: 'src/**/*.ts', expected: false },
      { glob: './src/**/*.ts', expected: false },
      { glob: '*.ts', expected: false },
      { glob: '{src,test}/**/*.ts', expected: false },
    ];
    cases.forEach(({ glob, expected }) => {
      then(`${glob} → ${expected}`, () => {
        expect(isUnboundGlob(glob)).toEqual(expected);
      });
    });
  });
});

describe('computePlanKey', () => {
  then('same inputs produce same key', () => {
    const key1 = computePlanKey({ old: 'foo', new: 'bar', glob: 'src/**/*.ts' });
    const key2 = computePlanKey({ old: 'foo', new: 'bar', glob: 'src/**/*.ts' });
    expect(key1).toEqual(key2);
  });

  then('different inputs produce different keys', () => {
    const key1 = computePlanKey({ old: 'foo', new: 'bar', glob: 'src/**/*.ts' });
    const key2 = computePlanKey({ old: 'foo', new: 'bar', glob: 'src/**/*.test.ts' });
    expect(key1).not.toEqual(key2);
  });
});

## docs updates

> **⚠️ CRITICAL** — these updates prevent shell expansion footguns.
> without single-quoted globs, shells expand `**/*.ts` before the command runs.
> this leads to silent failures or unexpected behavior. core part of the safety tune.

### rule.prefer.sedreplace-for-renames.md

update all examples to use single quotes:

```diff
- npx rhachet run --skill sedreplace --old "oldName" --new "newName" --glob "**/*.ts"
+ npx rhachet run --skill sedreplace --old "oldName" --new "newName" --glob 'src/**/*.ts'
```

> **note:** after update, re-run condense to update the `.min` file:
> `npx rhachet run --skill condense --from briefs/practices/work.flow/refactor/rule.prefer.sedreplace-for-renames.md --mode apply`
