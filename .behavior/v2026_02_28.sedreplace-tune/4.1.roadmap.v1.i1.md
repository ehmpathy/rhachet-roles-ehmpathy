## phase 1: core safety guards

### read before phase
- `.behavior/v2026_02_28.sedreplace-tune/3.3.blueprint.v1.i1.md` — contracts section
- `src/domain.roles/ergonomist/briefs/cli/rule.require.treestruct-output.md` — output format

### checklist

#### 1.1 glob validation
- [ ] add `isUnboundGlob` check after argument parse
- [ ] if glob starts with `**/` → print blocked message, exit 2
- [ ] use turtle vibes treestruct format for blocked output

#### 1.2 plan tracker
- [ ] create `.claude/sedreplace.nudges.local.json` if not exists
- [ ] compute plan key = `sha256(old|new|glob)`
- [ ] on `--mode plan`: record plan key with timestamp
- [ ] on `--mode apply`: check plan key exists, block if not
- [ ] cleanup stale entries (>1 hour old) on every invocation

#### 1.3 vibey treestruct output
- [ ] update plan mode output to use 🐢 header + 🐚 shell root
- [ ] update apply mode output to match
- [ ] each file gets its own sub.bucket with diff lines

### acceptance
- [ ] `--glob '**/*.ts'` → blocked with "unbound glob" message
- [ ] `--glob 'src/**/*.ts'` → allowed
- [ ] `--mode apply` without prior plan → blocked with "must plan first" message
- [ ] `--mode plan` followed by matched `--mode apply` → allowed
- [ ] output uses turtle vibes treestruct with sub.buckets

### verify
```sh
# test unbound glob block
npx rhachet run --skill sedreplace --old "foo" --new "bar" --glob '**/*.ts'
# expect: exit 2, "unbound glob" message

# test bounded glob allowed
npx rhachet run --skill sedreplace --old "foo" --new "bar" --glob 'src/**/*.ts'
# expect: exit 0, plan output

# test apply without plan
rm -f .claude/sedreplace.nudges.local.json
npx rhachet run --skill sedreplace --old "foo" --new "bar" --mode apply
# expect: exit 2, "must plan first" message

# test plan then apply
npx rhachet run --skill sedreplace --old "foo" --new "bar" --glob 'src/**/*.ts'
npx rhachet run --skill sedreplace --old "foo" --new "bar" --glob 'src/**/*.ts' --mode apply
# expect: both succeed
```
