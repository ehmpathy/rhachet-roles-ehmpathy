## pattern.1: output utils (turtle vibes)

**source**: `src/domain.roles/mechanic/skills/git.commit/output.sh` [1]

```bash
# print turtle emoji + phrase
# usage: print_turtle_header "cowabunga!"
print_turtle_header() {
  local phrase="${1:-}"
  echo ""
  echo "🐢 ${phrase}"
  echo ""
}

# print command name as tree root
# usage: print_tree_start "git.commit.set"
print_tree_start() {
  local cmd="$1"
  echo "🐚 ${cmd}"
}

# print tree branch (non-leaf)
# usage: print_tree_branch "commit"
print_tree_branch() {
  local label="$1"
  local indent="${2:-3}"
  printf "%${indent}s├─ %s\n" "" "$label"
}

# print tree leaf (terminal node)
# usage: print_tree_leaf "header" "fix(api): validate input"
print_tree_leaf() {
  local label="$1"
  local value="$2"
  local indent="${3:-3}"
  printf "%${indent}s│  └─ %s: %s\n" "" "$label" "$value"
}
```

**relation to wish**: provides consistent turtle vibes output format

**decision**: [REUSE] — source the output.sh file directly for git.branch.rebase skills

## pattern.3: plan/apply mode

**source**: `src/domain.roles/mechanic/skills/git.commit/git.commit.set.sh` [3]

```bash
# mode handling
MODE="plan"  # default to plan

while [[ $# -gt 0 ]]; do
  case $1 in
    --mode)
      MODE="$2"
      shift 2
      ;;
    # ...
  esac
done

# plan mode: preview only
if [[ "$MODE" == "plan" ]]; then
  # show preview
  print_turtle_header "heres the wave..."
  # ...
  echo ""
  echo "run with --mode apply to execute"
  exit 0
fi

# apply mode: execute
if [[ "$MODE" == "apply" ]]; then
  # execute action
  # ...
fi
```

**relation to wish**: git.branch.rebase begin uses plan mode by default, apply to execute

**decision**: [REUSE] — same plan/apply pattern for git.branch.rebase begin

# guard: cannot push to main/master
CURRENT_BRANCH=$(git rev-parse --abbrev-ref HEAD)

  print_turtle_header "hold up dude..."
