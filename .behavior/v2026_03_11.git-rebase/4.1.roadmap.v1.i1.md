### checklist

- [ ] create `git.branch.rebase.begin.sh` with:
  - [ ] source output.sh from git.commit
  - [ ] source git.branch.rebase.operations.sh
  - [ ] parse --mode argument (default: plan)
  - [ ] check push permission from meter
  - [ ] guard: reject if main/master
  - [ ] guard: reject if dirty work tree
  - [ ] guard: reject if rebase in progress
  - [ ] guard: report if already up to date
  - [ ] plan mode: fetch, preview commits/behind/meter
  - [ ] apply mode: git pull origin main --rebase
  - [ ] apply success: "righteous! ready to push"
  - [ ] apply conflict: show conflict files, guide to continue

- [ ] create `git.branch.rebase.begin.integration.test.ts` with:
  - [ ] [case1] plan mode shows preview
  - [ ] [case2] apply mode succeeds
  - [ ] [case3] apply mode with conflicts
  - [ ] [case4] no push permission
  - [ ] [case5] on main branch
  - [ ] [case6] dirty work tree
  - [ ] [case7] already up to date
  - [ ] [case8] rebase already in progress

### acceptance

```bash
# all tests pass
npm run test:integration -- git.branch.rebase.begin

# plan mode matches vision stdout demo
rhx git.branch.rebase begin # shows turtle vibes tree output

# apply mode works
rhx git.branch.rebase begin --mode apply # executes rebase
```
