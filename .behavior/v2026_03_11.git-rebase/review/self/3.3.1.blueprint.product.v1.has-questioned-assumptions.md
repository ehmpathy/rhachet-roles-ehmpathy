## assumption 1: git pull --rebase is the right command

**what we assume:** `git pull origin main --rebase` is correct for rebase onto main.

**what if opposite:** use `git rebase origin/main` directly.

**analysis:** `git pull --rebase` fetches AND rebases atomically. if we used `git rebase origin/main` alone, we'd need a separate fetch. the combined command is more robust.

**verdict:** assumption holds.

## fixes applied

### fix 1: detect default branch (main vs master)

**before:**
```bash
git fetch origin main 2>/dev/null
git pull origin main --rebase
```

**after:**
```bash
# detect default branch
if git rev-parse --verify origin/main &>/dev/null; then
  DEFAULT_BRANCH="main"
elif git rev-parse --verify origin/master &>/dev/null; then
  DEFAULT_BRANCH="master"
else
  echo "error: no main or master branch found on origin"
  exit 1
fi

git fetch origin "$DEFAULT_BRANCH"
git pull origin "$DEFAULT_BRANCH" --rebase
```

**files changed:**
- `src/domain.roles/mechanic/skills/git.rebase/git.rebase.begin.sh`

**rationale:** not all repos use "main". many still use "master". detect dynamically.

## assumption 7: origin/main is the base branch

**what we assume:** hardcode "origin/main" as rebase target.

**what if opposite:** repo uses "master" or "develop" as default branch.

**analysis:**
- `git fetch origin main` will fail if default branch is "master"
- `git pull origin main --rebase` will fail similarly
- many repos still use "master"

**could a simpler approach work?**

option 1: detect default branch dynamically
```bash
DEFAULT_BRANCH=$(git symbolic-ref refs/remotes/origin/HEAD 2>/dev/null | sed 's@^refs/remotes/origin/@@')
DEFAULT_BRANCH="${DEFAULT_BRANCH:-main}"
```

option 2: try main, fallback to master
```bash
if git rev-parse --verify origin/main &>/dev/null; then
  TARGET="origin/main"
elif git rev-parse --verify origin/master &>/dev/null; then
  TARGET="origin/master"
else
  echo "error: no main or master branch found"
  exit 1
fi
```

**decision:** option 2 is more robust and explicit.

**action needed:** update begin.sh to detect main vs master.
