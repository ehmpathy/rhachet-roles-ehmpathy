## unrelated failures in other tests

when the full integration suite runs, some brief.compress perfeval tests fail due to a corrupted cache file:

```
SyntaxError: Unexpected non-whitespace character after JSON at position 33208
  at JSON.parse (<anonymous>)
  at get (node_modules/.pnpm/simple-on-disk-cache@1.7.3/...)
```

### evidence this is pre-extant and unrelated

**git log shows brief.compress was not touched by this branch:**

```bash
$ git log --oneline -3 -- 'src/domain.roles/librarian/skills/brief.compress/'
81244e1 chore(roles): rename designer to ergonomist (#260)
cf196a9 fix(mechanic): simplify multiline commit messages and uses set args (#234)
0a06e65 feat(condensor): evals and pipelines for brief compression (#228)
```

all commits are on main, none from this branch (vlad/git-rebase).

**git diff shows this branch only touches git.branch.rebase files:**

```bash
$ git diff main --stat -- 'src/domain.roles/mechanic/skills/git.branch.rebase/'
 git.branch.rebase.abort.integration.test.ts     | 210 +++++
 git.branch.rebase.abort.sh                      |  65 ++
 git.branch.rebase.begin.integration.test.ts     | 370 +++++++++
 git.branch.rebase.begin.sh                      | 240 ++++++
 git.branch.rebase.continue.integration.test.ts  | 257 ++++++
 git.branch.rebase.continue.sh                   | 143 ++++
 git.branch.rebase.operations.sh                 | 167 ++++
 ...                                      | ...
```

**root cause is cache corruption, not code:**

the error trace shows:
- `simple-on-disk-cache/src/cache.ts:245:33` — cache read failed
- `JSON.parse` failed at position 33208 — corrupted JSON in cache file
- not a test assertion failure — the test never ran to completion

### why this is unrelated to git.branch.rebase

| attribute | brief.compress | git.branch.rebase |
|-----------|----------------|------------|
| domain | librarian | mechanic |
| directory | src/domain.roles/librarian/skills/ | src/domain.roles/mechanic/skills/ |
| failure type | cache corruption | n/a (all pass) |
| branch touch | no commits | all new files |

**disposition:** pre-extant cache corruption, not introduced by git.branch.rebase behavior.
