# distill: experience reproductions

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| verbose revoke sees tip | `git.commit.uses set --quant 0 --push block` | run command | stdout contains "revoked" + tip | integration |
| del shortcut works | `git.commit.uses del` | run command | stdout contains "revoked", no tip | integration |
| del is idempotent | `git.commit.uses del` | run when no quota | command succeeds, stdout contains "revoked" | integration |
| push-only has no tip | `git.commit.uses set --quant 0 --push allow` | run command | stdout contains "commits: 0", no tip | integration |
| short form revoke | `git.commit.uses set --quant 0` | run command (no --push) | --push defaults to block, shows tip | integration |

## reproduction feasibility

### test utilities available

- `runInTempGitRepo({ args, meterState? })` — creates temp git repo, runs skill, returns stdout/stderr/exitCode
- `given/when/then` from test-fns — BDD structure
- `expect().toMatchSnapshot()` — verify full output format
- `expect().toContain()` — verify specific strings

### setup required

- temp git repo (handled by fixture)
- optional .meter state (handled by fixture)

### test sketch

```typescript
given('[case4] del command', () => {
  when('[t0] quota was granted', () => {
    then('shows revoked without tip', () => {
      const result = runInTempGitRepo({
        args: ['del'],
        meterState: { uses: 3, push: 'allow' },
      });

      expect(result.exitCode).toBe(0);
      expect(result.stdout).toContain('revoked');
      expect(result.stdout).not.toContain('tip:');
      expect(result.stdout).toMatchSnapshot();
    });
  });

  when('[t1] no quota set', () => {
    then('still shows revoked (idempotent)', () => {
      const result = runInTempGitRepo({
        args: ['del'],
        // no meterState — quota not set
      });

      expect(result.exitCode).toBe(0);
      expect(result.stdout).toContain('revoked');
      expect(result.stdout).toMatchSnapshot();
    });
  });
});
```
