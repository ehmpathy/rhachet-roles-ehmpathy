## implementation details

### print_tip() function

```bash
print_tip() {
  local text="$1"
  # \033[2m = dim, \033[0m = reset
  echo -e "   └─ \033[2mtip: $text\033[0m"
}
```

### del case

```bash
del)
  QUANT=0
  PUSH="block"
  VIA_DEL=true
  ;& # fall through to set logic
```

### revoke output with tip

```bash
if [[ "$QUANT" == "0" && "$PUSH" == "block" ]]; then
  print_turtle_header "groovy, break time"
  print_tree_start "git.commit.uses ${VIA_DEL:+del}"
  if [[ -n "$VIA_DEL" ]]; then
    echo "   └─ revoked"
  else
    echo "   ├─ revoked"
    print_tip "'rhx git.commit.uses del' does the same"
  fi
```

## contracts

### input

| command | args | behavior |
|---------|------|----------|
| `del` | none | set quant=0, push=block, no tip |
| `set --quant 0` | `--push` optional | quant=0, push defaults to block, show tip |
| `set --quant 0 --push block` | explicit | quant=0, push=block, show tip |
| `set --quant 0 --push allow` | explicit | quant=0, push=allow, NO tip |

### output

| scenario | stdout |
|----------|--------|
| del | `🐚 git.commit.uses del` + `└─ revoked` |
| set revoke | `🐚 git.commit.uses set` + `├─ revoked` + `└─ tip: ...` |
| set push-only | `🐚 git.commit.uses set` + normal output (commits: 0, push: allowed) |
