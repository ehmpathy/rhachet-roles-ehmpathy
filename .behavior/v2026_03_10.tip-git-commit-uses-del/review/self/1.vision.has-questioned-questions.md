### question 3: what ANSI code for dim?

**source:** needed for implementation.

**can be answered via research:** yes.

**research:** ANSI SGR (Select Graphic Rendition) codes:
- `\033[2m` = dim/faint
- `\033[0m` = reset all attributes
- alternative: `\e[2m` and `\e[0m` (bash shorthand)

**example:**
```bash
echo -e "\033[2mtip: use 'del'\033[0m"
```

**answer:** `\033[2m` for dim, `\033[0m` for reset.

---

### question 4: should `del` delete file or set state?

**source:** ambiguous from wish "does the same"

**can be answered via logic:** yes.

**analysis:**
- `set --quant 0 --push block` writes `{uses: 0, push: "block"}` to file
- `del` should do the same (not delete file)
- if `del` deleted file, `get` would say "no quota set" instead of showing meter
- this would break UX

**answer:** `del` sets state to {uses: 0, push: "block"}, does not delete file.
