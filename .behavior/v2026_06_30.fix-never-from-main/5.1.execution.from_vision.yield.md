## defects found + fixed in execution

1. **`build:compile` does not rsync `.sh` into dist** — only `npm run build` (full)
   copies skill shell files. an intermediate test ran against stale dist. fix: always
   full-build before shell-skill tests.
2. **`tee /dev/stderr` fails in spawned (non-tty) contexts** — `tee: /dev/stderr:
   No such device or address` swallowed the guard message. fix: capture the message
   once into a var and emit to stdout + `>&2` explicitly (no `/dev/stderr` path).
   this also removed the `set -e` abort-before-`exit 2` hazard.
