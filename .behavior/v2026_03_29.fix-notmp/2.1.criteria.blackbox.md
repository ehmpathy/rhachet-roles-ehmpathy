# blackbox criteria: stop /tmp permission noise

## usecase.1 = read from claude temp directory

```
given(a mechanic wants to inspect claude session files)
  when(mechanic reads from /tmp/claude-{uid}/*)
    then(file contents shown immediately)
      sothat(investigation flows without friction)
    then(no permission prompt appears)
      sothat(mechanic focus is preserved)

  when(mechanic reads multiple files from /tmp/claude-{uid}/*)
    then(each file shown immediately)
    then(no permission prompts for any file)
      sothat(multi-file investigations flow smoothly)
```

## usecase.2 = read from non-claude temp paths

```
given(a mechanic wants to read files from /tmp not owned by claude)
  when(mechanic reads from /tmp/other-stuff)
    then(permission prompt appears)
      sothat(access to external temp files remains controlled)
```

## usecase.3 = write to /tmp (blocked)

```
given(a mechanic attempts to write scratch files)
  when(mechanic writes to /tmp/*)
    then(operation is blocked)
    then(guidance message explains why)
      sothat(mechanic understands /tmp is not actually temporary)
    then(guidance message suggests .temp/ alternative)
      sothat(mechanic knows the correct path forward)
    then(guidance shows example command)
      sothat(mechanic can immediately act on the guidance)

  when(mechanic writes to /tmp/claude-{uid}/*)
    then(operation is blocked)
      sothat(tool writes are prevented even to claude paths)
    then(guidance message appears)
      sothat(behavior is consistent for all /tmp writes)
```

## usecase.4 = write to .temp/ (allowed)

```
given(a mechanic needs scratch space)
  when(mechanic writes to .temp/*)
    then(file is created)
      sothat(scratch files stay in repo scope)
    then(no permission prompt appears)
      sothat(repo-local scratch is frictionless)
    then(.temp/ directory auto-created if absent)
      sothat(first write succeeds without setup)
```

## exchange boundaries

### read exchange

| input | output |
|-------|--------|
| `cat /tmp/claude-1000/...` | file contents, no prompt |
| `tail /tmp/claude-1000/...` | file contents, no prompt |
| `head /tmp/claude-1000/...` | file contents, no prompt |
| `cat /tmp/other-process/...` | permission prompt (external) |

### write exchange (blocked)

| input | output |
|-------|--------|
| `echo x > /tmp/scratch.txt` | blocked + guidance |
| `echo x > /tmp/claude-1000/...` | blocked + guidance |

guidance message content:
```
🛑 BLOCKED: /tmp is not actually temporary

/tmp persists indefinitely and never auto-cleans.
use .temp/ instead - it's scoped to this repo and gitignored.

  echo "data" > .temp/scratch.txt
```

### write exchange (allowed)

| input | output |
|-------|--------|
| `echo x > .temp/scratch.txt` | file created, no prompt |

## edge cases

| edge | expected behavior |
|------|-------------------|
| read from /tmp/claude* | auto-allowed |
| read from /tmp/not-claude* | permission prompt (correct) |
| write to /tmp/* | blocked with guidance |
| write to /tmp/claude* | blocked with guidance |
| write to .temp/* | allowed |
| .temp/ absent on write | auto-created |
| read tool vs bash tool | both auto-allowed for /tmp/claude* |
