# evaluation: /tmp permission behavior

## summary (as implemented)

implemented two-part solution:
1. **auto-allow reads** from `/tmp/claude*` — achieved via extant broad permissions (`Bash(cat:*)`, `Bash(head:*)`, `Bash(tail:*)`)
2. **auto-block writes** to `/tmp/*` via PreToolUse hook with guidance message
