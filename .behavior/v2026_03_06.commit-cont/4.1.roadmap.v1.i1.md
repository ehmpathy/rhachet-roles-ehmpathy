## phase 1: implement helper function

### tasks
- [ ] add `get_behavioral_commits_on_branch()` function to `git.commit.set.sh`

### tasks

- [ ] `fix:` after `fix:` is BLOCKED with error
- [ ] `feat:` after `fix:` is BLOCKED with error
- [ ] `cont:` after `fix:` SUCCEEDS

- [ ] error message shows first behavioral commit
