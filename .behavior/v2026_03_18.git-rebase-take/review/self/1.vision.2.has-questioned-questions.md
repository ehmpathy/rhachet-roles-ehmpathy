## question 4: partial failure behavior?

**from second review:**
> if glob matches 3 files in conflict, 2 succeed, 1 fails — what do we do?

**can this be answered via logic now?**
yes — best-effort with clear error report is better for this use case.

**rationale:**
- settlement is interactive, not transactional
- clone can see what succeeded and fix what failed
- atomic rollback would undo progress, waste effort

**answer:** best-effort with error report at end.

**status:** [answered] — add to vision
