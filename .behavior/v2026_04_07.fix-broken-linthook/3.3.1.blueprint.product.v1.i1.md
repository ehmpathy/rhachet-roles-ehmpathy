# blueprint: git.repo.test skill

## summary

create a new skill `git.repo.test` that runs lint checks with:
- exit code 2 on failure to enforce brain action
- turtle vibes summary output (not raw lint output)
- log files for full lint output
- findsert .gitignore in log directory

replace the `pnpm run --if-present fix` hook in mechanic onStop hooks.

## test coverage

### integration tests (git.repo.test.integration.test.ts)

```
usecase.1 = lint passes
├─ given: temp repo with package.json + test:lint that passes
├─ when: rhx git.repo.test --what lint
└─ then: exit 0, stdout contains success summary

usecase.2 = lint fails
├─ given: temp repo with lint errors
├─ when: rhx git.repo.test --what lint
└─ then: exit 2, stdout contains failure summary + defect count + log path + tip

usecase.3 = npm error
├─ given: temp repo with broken test:lint command
├─ when: rhx git.repo.test --what lint
└─ then: exit 1, stderr contains error

usecase.4 = no package.json
├─ given: temp repo without package.json
├─ when: rhx git.repo.test --what lint
└─ then: exit 2, explains absent package.json

usecase.5 = log directory findsert
├─ given: temp repo with lint (log dir does not exist)
├─ when: rhx git.repo.test --what lint
└─ then: log dir created, .gitignore created with self-ignore

usecase.6 = log file content
├─ given: temp repo with lint errors
├─ when: rhx git.repo.test --what lint
└─ then: log file contains full lint output
```

## contracts

### skill input

```
--what lint                                    # required (only lint supported)
--when <context>                               # optional, for future use
```

### skill output

```
stdout: turtle vibes summary
  ├─ status (passed/failed)
  ├─ defects (count, if failed)
  ├─ log (path)
  └─ tip (npm run fix)

stderr: empty (lint output goes to log file)

exit code:
  0 = passed
  1 = malfunction (npm error)
  2 = constraint (lint failed)
```

### log files

```
.log/role=mechanic/skill=git.repo.test/{isotime}.stdout.log
.log/role=mechanic/skill=git.repo.test/{isotime}.stderr.log
.log/role=mechanic/skill=git.repo.test/.gitignore  # findsert, self-ignore
```
