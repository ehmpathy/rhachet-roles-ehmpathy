# blackbox criteria: git.repo.test --what lint

## usecase.1 = lint check passes

```
given([case1] repo with lint that passes)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(exit code is 0)
      sothat(brain knows no action needed)
    then(stdout shows turtle success summary)
      sothat(mechanic sees status at a glance)
    then(stdout shows "status: passed")
    then(stdout shows log path)
      sothat(mechanic can review full output if curious)
    then(stderr is empty)
      sothat(tokens are not wasted on raw lint output)
```

## usecase.2 = lint check fails

```
given([case1] repo with lint defects)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(exit code is 2)
      sothat(brain is forced to address the constraint)
    then(stdout shows turtle failure summary)
      sothat(mechanic sees status at a glance)
    then(stdout shows "status: failed")
    then(stdout shows defect count)
      sothat(mechanic knows scope of problem)
    then(stdout shows log path)
      sothat(mechanic can review defect details)
    then(stdout shows tip to try `npm run fix`)
      sothat(mechanic knows the quick fix path)
    then(stderr is empty)
      sothat(tokens are not wasted on raw lint output)
```

## usecase.3 = lint command fails (malfunction)

```
given([case1] repo where `npm run test:lint` errors out)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(exit code is 1)
      sothat(brain sees malfunction, not constraint)
    then(stdout shows error summary)
      sothat(mechanic sees what went wrong)
    then(stderr captures the npm error)
      sothat(error details are available)
```

## usecase.4 = no package.json

```
given([case1] directory without package.json)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(exit code is 1)
      sothat(brain sees malfunction)
    then(stdout explains absent package.json)
      sothat(mechanic knows why it failed)
```

## usecase.5 = log directory management

```
given([case1] repo where .log/role=mechanic/skill=git.repo.test/ does not exist)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(log directory is created)
    then(.gitignore is created in log directory)
      sothat(logs are not committed to git)
    then(.gitignore contains self-ignore pattern)
      sothat(the .gitignore itself is also ignored)

given([case2] repo where log directory already exists with .gitignore)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(.gitignore is unchanged)
      sothat(findsert semantics are honored)
```

## usecase.6 = context hint (--when)

```
given([case1] any repo state)
  when([t0] `rhx git.repo.test --what lint --when hook.onStop` is run)
    then(behavior is identical to without --when)
      sothat(context hint is for future use only)
```

## usecase.7 = log file content

```
given([case1] repo with lint defects)
  when([t0] `rhx git.repo.test --what lint` is run)
    then(log file contains full npm stdout)
      sothat(defect details are preserved)
    then(log file contains full npm stderr)
      sothat(error context is preserved)
    then(log filename includes isotime)
      sothat(multiple runs are distinguishable)
```
