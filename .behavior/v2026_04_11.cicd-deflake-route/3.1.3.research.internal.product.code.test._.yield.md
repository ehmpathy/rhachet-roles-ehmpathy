## pattern.2 = shell skill test via spawnSync

**[REUSE]** shell skills are tested via `spawnSync('bash', [scriptPath, ...args])`.

### citation.2 — spawnSync pattern for shell skills

> source: `src/domain.roles/mechanic/skills/declapract.upgrade.integration.test.ts:40-66`
>
> ```typescript
> const runSkill = (options: {
>   cwd: string;
>   subcommand?: string;
>   args?: string[];
>   env?: Record<string, string>;
> }): { stdout: string; stderr: string; status: number | null } => {
>   const allArgs = options.subcommand
>     ? [options.subcommand, ...(options.args ?? [])]
>     : (options.args ?? []);
>
>   const result = spawnSync('bash', [SKILL_PATH, ...allArgs], {
>     cwd: options.cwd,
>     encoding: 'utf-8',
>     env: {
>       ...process.env,
>       SKIP_ROUTE_BIND: '1',
>       ...options.env,
>     },
>   });
>
>   return {
>     stdout: result.stdout ?? '',
>     stderr: result.stderr ?? '',
>     status: result.status,
>   };
> };
> ```

**relation to wish**: cicd.deflake init tests will use same spawnSync pattern.
