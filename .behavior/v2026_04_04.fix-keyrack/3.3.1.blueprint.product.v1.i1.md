## contracts

### keyrack SDK usage

```typescript
import { keyrack } from 'rhachet/keyrack';

const result = await keyrack.get({
  for: { key: 'XAI_API_KEY' },
  owner: 'ehmpath',
  env: 'prep',
});

if (result.attempt.status !== 'granted') {
  console.error(result.emit.stdout);
  process.exit(2);
}

process.env.XAI_API_KEY = result.attempt.grant.key.secret;
```
