describe('cicd.deflake', () => {

## 3. runSkill utility pattern

**[REUSE]** the runSkill utility for skill invocation in tests

```typescript
// src/domain.roles/mechanic/skills/cicd.deflake/__test_utils__/runSkill.ts [3]
import { spawnSync } from 'child_process';

export const runSkill = (options: {
  cwd: string;
  subcommand?: string;
  args?: string[];
  env?: Record<string, string>;
}): { stdout: string; stderr: string; status: number | null } => {
  const result = spawnSync('bash', [SKILL_PATH, ...allArgs], {
    cwd: options.cwd,
    encoding: 'utf-8',
    env: { ...process.env, ...options.env },
  });

  return {
    stdout: result.stdout ?? '',
    stderr: result.stderr ?? '',
    status: result.status,
  };
};
```

**relation to wish:** eval tests need to invoke focus skills and capture output. this pattern will be adapted for `runFocusSkill`.
