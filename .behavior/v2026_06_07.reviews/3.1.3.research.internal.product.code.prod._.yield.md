## 7. bhrain review skill dispatch

**[REUSE]** the review skill from bhrain

from the handoff document [8]:
```yaml
# architecture
rhx review.by --role mechanic
    │
    ├─ lookup role package path
    │     └─ .agent/repo=ehmpathy/role=mechanic/
    │
    ├─ load focuses from role
    │     └─ briefs/reviews/focuses.yml
    │
    └─ for each focus:
          └─ rhx review --rules $focus.rules --paths $paths --output $output
```

**relation to wish:** this is the architecture we'll implement. focus skills wrap `rhx review`.
