## implementation notes

### key changes to rmsafe.sh

1. **helper function**: `is_gitignored()`
   - uses `git check-ignore -q` to check if path is ignored
   - returns 0 if ignored, 1 if trackable

2. **file removal logic**:
   - if gitignored → `rm` directly, no trash copy
   - if trackable → current behavior (copy to trash, then rm)

3. **directory removal logic**:
   - enumerate all files in directory
   - for each file: check if gitignored
   - trackable files → copy to trash
   - gitignored files → skip trash
   - then `rm -rf` the directory

4. **output changes**:
   - show "(gitignored)" in type field for direct deletes
   - only show coconut hint if any trackable files were trashed
