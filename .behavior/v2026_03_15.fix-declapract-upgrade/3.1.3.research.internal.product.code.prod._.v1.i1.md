## pattern 1: subcommand routing [NEEDED]

the extant skill has no subcommand routing. it runs `init` behavior directly.

**citation 1** — `init.declapract.upgrade.sh` lines 31-63:
```bash
while [[ $# -gt 0 ]]; do
  case $1 in
    # rhachet passes these - ignore them
    --skill|--repo|--role)
      shift 2
      ;;
    --help|-h)
      echo "usage: rhx init.declapract.upgrade"
      ...
      exit 0
      ;;
    *)
      echo "unknown argument: $1"
      exit 1
      ;;
  esac
done
```

no subcommand parsing. for `declapract.upgrade init|exec`, we need:
- parse first positional arg as subcommand
- route to appropriate handler
- fail with usage on invalid subcommand

**ref pattern** — `git.commit.uses.sh` does subcommand routing (seen in boot briefs)
