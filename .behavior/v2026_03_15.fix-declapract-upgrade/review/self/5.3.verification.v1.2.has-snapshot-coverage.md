## why exec output lacks snapshot

exec runs multiple external commands:
- fnm use
- pnpm add
- npx declapract apply (twice)
- pnpm install
- pnpm fix

this output is:
1. verbose (hundreds of lines)
2. non-deterministic (timestamps, versions)
3. external tool output (not our contract)

snapshot would be noisy and fragile.

## conclusion

snapshot coverage is intentional:
- **init**: has snapshot (user-faced output matters)
- **exec**: no snapshot (external tool output, too noisy)
- **errors**: no snapshot (simple strings, assertions sufficient)

the gate holds.
