## friction analysis

### init happy path

```bash
rhx declapract.upgrade init
```

**friction points:**
- requires declapract.use.yml (clear error if absent)
- creates route directory
- copies templates
- binds to branch

**smooth?** yes — one command, clear output, clear error

### exec happy path

```bash
rhx declapract.upgrade exec
```

**friction points:**
- requires declapract.use.yml (clear error if absent)
- requires pnpm (clear error if absent)
- runs multiple external commands

**smooth?** yes — one command, verbose but complete output

### exec standalone

```bash
rhx declapract.upgrade exec
```

**friction points:** same as exec happy

**smooth?** yes — no dependency on prior init

## conclusion

all critical paths are single-command invocations with clear error messages.

the gate holds.
