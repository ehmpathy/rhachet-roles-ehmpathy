# blueprint: declapract.upgrade refactor

## summary

refactor `init.declapract.upgrade` skill into `declapract.upgrade` skill with two subcommands:
- `declapract.upgrade init` — creates route, binds to branch (no .sh files)
- `declapract.upgrade exec` — runs upgrade logic directly from skill

key changes:
- eliminate chmod 555 management
- eliminate .sh file copy to route
- stone 1 instructs user to run `rhx declapract.upgrade exec`

## key contracts

### declapract.upgrade.sh

```
input: subcommand (init|exec)
output: route (init) or upgraded repo (exec)
exit codes:
  0 = success
  1 = invalid args
  2 = prereq failure (no declapract.use.yml, no pnpm)
```

### declapract.upgrade/init.sh

```
prereqs:
  - declapract.use.yml exists
output:
  - .route/v$isodate.declapract.upgrade/ with stones/guards
  - route bound to current branch
idempotency: findsert (overwrites extant route)
```

### declapract.upgrade/exec.sh

```
prereqs:
  - declapract.use.yml exists
  - pnpm available
output:
  - declapract packages upgraded
  - practices applied (twice for idempotency)
  - dependencies reinstalled
  - lint/format fixed
idempotency: yes (declapract apply is idempotent)
```
