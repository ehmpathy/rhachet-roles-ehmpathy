# evaluation: declapract.upgrade (as implemented)

## summary (as implemented)

refactored `init.declapract.upgrade` skill into `declapract.upgrade` skill with two subcommands:
- `declapract.upgrade init` — creates route, binds to branch (no .sh files)
- `declapract.upgrade exec` — runs upgrade logic directly from skill

key changes:
- eliminated chmod 555 management
- eliminated .sh file copy to route (whitelist pattern: only .stone and .guard files copied)
- stone 1 instructs user to run `rhx declapract.upgrade exec`
