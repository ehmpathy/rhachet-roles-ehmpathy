# blackbox criteria: declapract.upgrade

## usecase.1 = init a declapract upgrade route

```
given(a repo with declapract.use.yml)
  when(user runs `rhx declapract.upgrade init`)
    then(route directory created at .route/v$isodate.declapract.upgrade/)
      sothat(user has a structured workflow to follow)
    then(route bound to current branch)
      sothat(route.drive knows which route to use)
    then(turtle vibes output shown)
      sothat(user feels confident the skill ran)

given(a repo WITHOUT declapract.use.yml)
  when(user runs `rhx declapract.upgrade init`)
    then(error: "not a declapract repo")
      sothat(user knows to run `npx declapract init` first)
```

## usecase.2 = execute a declapract upgrade

```
given(a repo with declapract.use.yml)
  when(user runs `rhx declapract.upgrade exec`)
    then(node version set via fnm)
    then(declapract packages upgraded to latest)
    then(declapract practices applied twice)
      sothat(idempotency is verified)
    then(dependencies reinstalled)
    then(auto-fix run)
    then(success output shown)
      sothat(user knows upgrade completed)

given(a repo WITHOUT declapract.use.yml)
  when(user runs `rhx declapract.upgrade exec`)
    then(error: "not a declapract repo")

given(a repo without pnpm available)
  when(user runs `rhx declapract.upgrade exec`)
    then(error: "pnpm not found")
      sothat(user knows to install pnpm)
```

## usecase.3 = exec works standalone (no init required)

```
given(a repo with declapract.use.yml)
  given(no route has been created)
    when(user runs `rhx declapract.upgrade exec`)
      then(upgrade runs successfully)
        sothat(user can do quick one-off upgrades without ceremony)

given(a repo with declapract.use.yml)
  given(route already exists from previous init)
    when(user runs `rhx declapract.upgrade exec`)
      then(upgrade runs successfully)
        sothat(exec is idempotent regardless of route state)
```

## usecase.4 = init is idempotent

```
given(a repo with declapract.use.yml)
  given(route already exists from previous init)
    when(user runs `rhx declapract.upgrade init`)
      then(route directory overwritten with fresh templates)
        sothat(user can restart the workflow if needed)
      then(route bound to current branch)
      then(success output shown)
```

## usecase.5 = route.drive guides through stones

```
given(route created via `rhx declapract.upgrade init`)
  when(user runs `rhx route.drive`)
    then(user guided to 1.upgrade.invoke stone)
      sothat(user knows to run `rhx declapract.upgrade exec`)

given(1.upgrade.invoke stone passed)
  when(user runs `rhx route.drive`)
    then(user guided to 2.detect.hazards stone)
      sothat(user reviews what might have broken)

given(all stones passed)
  when(user runs `rhx route.drive`)
    then(route marked complete)
      sothat(user knows the upgrade workflow is done)
```
