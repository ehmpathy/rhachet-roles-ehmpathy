**problem**: git.branch.rebase.sh uses `SKILL_DIR` but git.commit.uses.sh uses `SCRIPT_DIR`.

| file | variable name |
|------|---------------|
| git.commit.uses.sh:27 | SCRIPT_DIR |

| git.branch.rebase.sh:24 | SKILL_DIR |

**analysis**: two conventions extant:
- `git.commit/*` uses `SCRIPT_DIR`
- `git.branch.rebase/*` uses `SKILL_DIR`

both mean the same thing - the directory of the skill.

**verdict**: within git.branch.rebase/ namespace, `SKILL_DIR` is consistent. the dispatcher follows the convention of its own namespace. no change needed.
