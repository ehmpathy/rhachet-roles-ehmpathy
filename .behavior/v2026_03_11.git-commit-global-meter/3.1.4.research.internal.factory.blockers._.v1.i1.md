# research: factory blockers

## test infra

### blocker: isolated `$HOME` for global storage tests

**status**: solvable, not a blocker

**issue**: global blocker writes to `~/.rhachet/storage/repo=ehmpathy/role=mechanic/.meter/`. tests cannot pollute the real `$HOME`.

**solution**: inject custom `HOME` env var in tests:
```typescript
const result = spawnSync('bash', [skillPath, ...args.args], {
  cwd: tempDir,
  env: { ...process.env, HOME: tempHomeDir },
  encoding: 'utf-8',
  stdio: ['pipe', 'pipe', 'pipe'],
});
```

**verification**: extant `spawnSync` pattern already supports `env` override.
