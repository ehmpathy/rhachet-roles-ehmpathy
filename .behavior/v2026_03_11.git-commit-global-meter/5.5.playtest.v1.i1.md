# playtest: global git.commit meter

## summary

manual verification of global circuit breaker for git.commit quota.

---

## prerequisites

1. build the project:
   ```sh
   npm run build
   ```

2. link skills:
   ```sh
   npx rhachet roles link --role mechanic
   ```

---

## playtest scenarios

### scenario 1: block commits globally

**goal**: verify `block --global` creates a global blocker.

```sh
# check current state
npx rhachet run --skill git.commit.uses get --global

# activate global blocker
npx rhachet run --skill git.commit.uses block --global
```

**expected output**:
```
🐢 groovy, bond fire time

🐚 git.commit.uses block --global
   └─ commits blocked globally
```

**verify blocker file created**:
```sh
cat ~/.rhachet/storage/repo=ehmpathy/role=mechanic/.meter/git.commit.uses.jsonc
```

**expected**: `{ "blocked": true }`

---

### scenario 2: check global state

**goal**: verify `get --global` shows blocker state.

```sh
# with blocker active (from scenario 1)
npx rhachet run --skill git.commit.uses get --global
```

**expected output**:
```
🐢 lets check the meter...

🐚 git.commit.uses get --global
   └─ global: blocked
```

---

### scenario 3: allow commits globally (lift blocker)

**goal**: verify `allow --global` removes the blocker.

```sh
npx rhachet run --skill git.commit.uses allow --global
```

**expected output**:
```
🐢 shell yeah, back in the water!

🐚 git.commit.uses allow --global
   └─ commits resumed globally
```

**verify blocker file removed**:
```sh
cat ~/.rhachet/storage/repo=ehmpathy/role=mechanic/.meter/git.commit.uses.jsonc
```

**expected**: file not found (or absent)

---

### scenario 4: local get shows global state

**goal**: verify local `get` shows global blocker status when active.

```sh
# grant local quota first
npx rhachet run --skill git.commit.uses set --quant 3 --push allow

# activate global blocker
npx rhachet run --skill git.commit.uses block --global

# check local state (should show global: blocked)
npx rhachet run --skill git.commit.uses get
```

**expected output** (includes global line):
```
🐢 lets check the meter...

🐚 git.commit.uses
   └─ meter
      ├─ left: 3
      ├─ push: allowed
      └─ global: blocked
```

---

### scenario 5: commit blocked by global blocker

**goal**: verify commits fail when global blocker is active (even with local quota).

```sh
# ensure global blocker is active
npx rhachet run --skill git.commit.uses block --global

# ensure local quota exists
npx rhachet run --skill git.commit.uses set --quant 3 --push allow

# create a test change
echo "test" >> /tmp/test-file.txt

# attempt to commit (this should be blocked)
# (run in a test repo with staged changes)
printf 'fix(test): test commit\n\n- test line' | npx rhachet run --skill git.commit.set -m @stdin
```

**expected**: exit code 2 with message:
```
🐢 bummer dude...

🐚 git.commit.set
   └─ error: commits blocked globally

ask your human to lift:
  $ git.commit.uses allow --global
```

---

### scenario 6: commit succeeds after global blocker lifted

**goal**: verify commits work after `allow --global`.

```sh
# lift global blocker
npx rhachet run --skill git.commit.uses allow --global

# attempt commit (should succeed if local quota present)
printf 'fix(test): test commit\n\n- test line' | npx rhachet run --skill git.commit.set -m @stdin
```

**expected**: commit succeeds, quota decrements.

---

### scenario 7: block/allow local (symmetric commands)

**goal**: verify local `block` and `allow` work as expected.

```sh
# block local (revokes quota)
npx rhachet run --skill git.commit.uses block
```

**expected output**:
```
🐢 groovy, break time

🐚 git.commit.uses del
   └─ revoked
```

```sh
# allow local (grants unlimited)
npx rhachet run --skill git.commit.uses allow
```

**expected output**:
```
🐢 radical! let's ride!

🐚 git.commit.uses set
   ├─ granted: unlimited
   └─ push: allowed
```

## edge cases

| edge case | how to test | expected |
|-----------|-------------|----------|
| block --global twice | run `block --global` twice | idempotent, no error |
| allow --global when not blocked | run `allow --global` without blocker | idempotent, no error |
| corrupt global file | manually corrupt the file | commit blocked, error about corrupt file |
| no local quota + global blocked | `del` local, `block --global` | get shows "no quota set" and "global: blocked" |
