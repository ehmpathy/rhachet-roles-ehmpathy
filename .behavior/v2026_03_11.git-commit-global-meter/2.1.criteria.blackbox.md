# blackbox criteria: global git.commit meter

## usecase.1 = block commits globally

```
given('human has granted quota in multiple repos')
  when('human runs `git.commit.uses block --global`')
    then('stdout shows turtle header with chill phrase')
    then('stdout shows `commits blocked globally`')
    then('global blocker file is written to ~/.rhachet/storage/repo=ehmpathy/role=mechanic/.meter/git.commit.uses.jsonc')
      sothat('all repos respect the global blocker')

given('global blocker is active')
  when('mechanic runs `git.commit.set` in any repo')
    then('stdout shows `bummer dude...`')
    then('stdout shows `error: commits blocked globally`')
    then('stdout shows instruction to ask human to lift')
    then('exit code is non-zero')
      sothat('commit is prevented')

given('global blocker is active')
  when('mechanic runs `git.commit.uses get` in a repo with local quota')
    then('stdout shows local quota (left, push)')
    then('stdout shows `global: blocked`')
      sothat('mechanic understands why commits are blocked')
```

## usecase.2 = allow commits globally (lift blocker)

```
given('global blocker is active')
  when('human runs `git.commit.uses allow --global`')
    then('stdout shows turtle header with chill phrase')
    then('stdout shows `commits resumed globally`')
    then('global blocker file is removed')
      sothat('repos return to local-only behavior')

given('global blocker was just lifted')
  when('mechanic runs `git.commit.set` in repo with local quota')
    then('commit proceeds normally')
    then('local quota decrements')
      sothat('local grants are preserved through global block/allow cycle')
```

## usecase.3 = block commits locally (per-repo)

```
given('human is in a git repo')
  when('human runs `git.commit.uses block`')
    then('stdout shows `groovy, break time`')
    then('stdout shows `revoked`')
    then('local quota is set to 0')
      sothat('mechanic cannot commit in this repo')
```

## usecase.4 = allow commits locally (per-repo)

```
given('human is in a git repo')
  when('human runs `git.commit.uses allow`')
    then('stdout shows `radical! let\'s ride!`')
    then('stdout shows `granted: unlimited`')
    then('stdout shows `push: allowed`')
      sothat('mechanic can commit freely in this repo')
```

## usecase.5 = check global state

```
given('global blocker is active')
  when('human runs `git.commit.uses get --global`')
    then('stdout shows global blocker is active')
      sothat('human can verify global state from anywhere')

given('no global blocker')
  when('human runs `git.commit.uses get --global`')
    then('stdout shows no global blocker')
      sothat('human can confirm repos use local-only behavior')
```

## usecase.6 = global blocker overrides local allow

```
given('repo has local quota (e.g., 3 uses, push allowed)')
  given('global blocker is active')
    when('mechanic attempts to commit')
      then('commit is blocked')
      then('local quota is NOT decremented')
        sothat('global blocker takes precedence')

given('repo has local quota')
  given('global blocker is lifted')
    when('mechanic commits')
      then('commit succeeds')
      then('local quota decrements')
        sothat('local grants resume after global blocker is lifted')
```

## usecase.7 = corrupt global file fails safe

```
given('global blocker file exists but is corrupt (invalid json)')
  when('mechanic attempts to commit')
    then('commit is blocked')
    then('error message indicates file issue')
      sothat('mechanic is blocked until human fixes the file')
```
