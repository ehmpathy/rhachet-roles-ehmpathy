# distillation: experience reproductions

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| block globally | `git.commit.uses block --global` | human runs command | stdout shows confirmation, file created | integration |
| allow globally | `git.commit.uses allow --global` | human runs command | stdout shows confirmation, file deleted | integration |
| block locally | `git.commit.uses block` | human runs command in repo | stdout shows revoked, quota set to 0 | integration |
| allow locally | `git.commit.uses allow` | human runs command in repo | stdout shows granted unlimited | integration |
| commit blocked globally | `git.commit.set` | mechanic tries commit with global blocker | exit 2, stdout shows error | integration |
| get with global blocker | `git.commit.uses get` | mechanic checks quota | shows local + "global: blocked" | integration |
| get global state | `git.commit.uses get --global` | human checks global | shows blocked/not blocked | integration |
| global commands outside repo | `git.commit.uses block --global` | run from non-git directory | command succeeds | integration |

### setup required

```typescript
const runWithGlobalStorage = (args: {
  args: string[];
  meterState?: { uses: number; push: string };
  globalBlocker?: boolean;
}): TestResult => {
  // create temp directories
  const tempDir = genTempDir({ slug: 'git-commit-uses-test', git: true });
  const tempHome = genTempDir({ slug: 'git-commit-uses-home', git: false });

  // set up local meter state if provided
  if (args.meterState) {
    const meterDir = path.join(tempDir, '.meter');
    fs.mkdirSync(meterDir, { recursive: true });
    fs.writeFileSync(
      path.join(meterDir, 'git.commit.uses.jsonc'),
      JSON.stringify(args.meterState, null, 2),
    );
  }

  // set up global blocker if provided
  if (args.globalBlocker) {
    const globalDir = path.join(
      tempHome,
      '.rhachet/storage/repo=ehmpathy/role=mechanic/.meter',
    );
    fs.mkdirSync(globalDir, { recursive: true });
    fs.writeFileSync(
      path.join(globalDir, 'git.commit.uses.jsonc'),
      JSON.stringify({ blocked: true }, null, 2),
    );
  }

  // run skill with injected HOME
  const result = spawnSync('bash', [skillPath, ...args.args], {
    cwd: tempDir,
    env: { ...process.env, HOME: tempHome },
    encode: 'utf-8',
  });

  return {
    stdout: result.stdout ?? '',
    stderr: result.stderr ?? '',
    exitCode: result.status ?? 1,
    tempDir,
    tempHome,
  };
};
```

### test sketch: global blocker blocks commits

```typescript
given('[case1] global blocker is active', () => {
  when('[t0] mechanic attempts commit', () => {
    then('commit is blocked with error', () => {
      const result = runWithGlobalStorage({
        args: ['set', '-m', 'test commit'],
        meterState: { uses: 3, push: 'allow' },
        globalBlocker: true,
      });

      expect(result.exitCode).toBe(2);
      expect(result.stdout).toContain('bummer dude');
      expect(result.stdout).toContain('commits blocked globally');
      expect(result.stdout).toMatchSnapshot();
    });
  });
});
```
