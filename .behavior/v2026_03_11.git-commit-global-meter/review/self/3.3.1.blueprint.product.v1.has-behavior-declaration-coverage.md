| stdout shows `push: allowed` | extant set output | ✅ |

### gap found: "unlimited" vs "999999"

**criterion**: `then('stdout shows \`granted: unlimited\`')`

**blueprint says**: `set --quant 999999 --push allow`

**extant output** (git.commit.uses.sh line 179):
```bash
echo "   ├─ granted: $QUANT"  # would show "granted: 999999"
```

**options**:
1. change output to detect 999999 and display "unlimited"
2. update criteria to say "granted: 999999"

**decision**: option 1 is better UX. add special case:
```bash
if [[ "$QUANT" == "999999" ]]; then
  echo "   ├─ granted: unlimited"
else
  echo "   ├─ granted: $QUANT"
fi
```
