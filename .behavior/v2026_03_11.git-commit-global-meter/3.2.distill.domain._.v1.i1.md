# distillation: domain model

## conceptual entities

### CommitQuotaLocal

**what**: per-repo quota state for mechanic commits.

**location**: `$GIT_REPO/.meter/git.commit.uses.jsonc`

**shape**:
```json
{
  "uses": 3,
  "push": "allow"
}
```

**attributes**:
| attribute | type | meaning |
|-----------|------|---------|
| `uses` | number | commits left (0 = revoked) |
| `push` | `"allow"` \| `"block"` | push permission |

**lifecycle**:
- created by `set --quant N --push allow|block`
- updated by `set` (overwrite) or commit (decrement)
- deleted by `del` or `block` (sets to 0)

---

### CommitBlockerGlobal

**what**: global circuit breaker that overrides all local quotas.

**location**: `~/.rhachet/storage/repo=ehmpathy/role=mechanic/.meter/git.commit.uses.jsonc`

**shape**:
```json
{
  "blocked": true
}
```

**attributes**:
| attribute | type | meaning |
|-----------|------|---------|
| `blocked` | `true` | global blocker is active |

**lifecycle**:
- created by `block --global`
- deleted by `allow --global`
- file absence = no global blocker

**precedence**: global blocker always wins over local quota.

---

## operations

### block --global

**effect**: creates CommitBlockerGlobal file.

**idempotent**: yes (re-run = same state).

---

### allow --global

**effect**: deletes CommitBlockerGlobal file.

**idempotent**: yes (file absent = no-op).

---

### block (local)

**effect**: sets CommitQuotaLocal to `{ uses: 0, push: "block" }`.

**alias**: same as `del`.

---

### allow (local)

**effect**: sets CommitQuotaLocal to `{ uses: @any, push: "allow" }`.

**note**: `@any` = sentinel for unlimited.

---

### get

**behavior**: reads both local and global state, shows effective.

**output logic**:
```
if global blocker exists:
  show local quota (if any)
  show "global: blocked"
else:
  show local quota only
```

---

### get --global

**behavior**: reads global state only.

**output logic**:
```
if global blocker exists:
  show "blocked"
else:
  show "not blocked"
```

---

## decision flow: can mechanic commit?

```
check global blocker
  ├─ exists → BLOCKED (global circuit breaker)
  └─ absent → check local quota
                ├─ has uses > 0 → ALLOWED
                └─ no uses or no file → BLOCKED (local)
```

## implementation notes

### shell-native

- no TypeScript domain objects — shell skill only
- `jq` used to parse and write JSON
- file existence checks via `[[ -f "$path" ]]`

### paths

| entity | path |
|--------|------|
| CommitQuotaLocal | `$GIT_REPO/.meter/git.commit.uses.jsonc` |
| CommitBlockerGlobal | `$HOME/.rhachet/storage/repo=ehmpathy/role=mechanic/.meter/git.commit.uses.jsonc` |

### constants

```bash
ROLE_REPO="ehmpathy"
ROLE_SLUG="mechanic"
GLOBAL_STORAGE_DIR="$HOME/.rhachet/storage/repo=$ROLE_REPO/role=$ROLE_SLUG"
GLOBAL_METER_FILE="$GLOBAL_STORAGE_DIR/.meter/git.commit.uses.jsonc"
```
