the playtest has path 3 for `--to prod` in plan mode but no path for apply mode. the vision shows this as a key usecase:

> run full release: merge → main CI → tag → deploy → `git.release --to prod --mode apply`

**fix**: add path 4 for prod apply mode.

### gap 2: no "needs rebase" edge path


**fix**: add edge 5 for needs rebase state.
