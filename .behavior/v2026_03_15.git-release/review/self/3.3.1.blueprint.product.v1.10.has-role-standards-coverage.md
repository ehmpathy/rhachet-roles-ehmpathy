## methodology

### approach taken

this coverage review followed these steps:

1. **enumerate rule directories** — same 10 directories as adherance review
2. **formulate coverage questions** — "is pattern X present?" for each category
3. **search blueprint for evidence** — line-by-line check of each section
4. **examine potential gaps** — proactive search for omissions
5. **cross-reference to vision/criteria** — verify blueprint matches spec

### what distinguishes coverage from adherance

| adherance review | coverage review |
|-----------------|-----------------|
| "does X violate rule Y?" | "is pattern Y present?" |
| looks for deviations | looks for omissions |
| asks "is this wrong?" | asks "is this complete?" |
| finds anti-patterns | finds gaps |

### how coverage gaps were examined

for each potential gap:
1. formulate question about what might be absent
2. search blueprint, vision, and criteria for evidence
3. if found, document the coverage
4. if not found, flag as gap and fix blueprint

thorough coverage review requires proactive search for omissions. the 10 gaps examined cover:
