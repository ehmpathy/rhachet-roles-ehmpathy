## gaps found

### gap 1: no `--retry` flag coverage

the wish and vision mention `--retry` to rerun failed workflows:

> hint: use --retry to rerun failed workflows

the playtest does not include a path for `--retry`.

**fix**: add edge 6 for retry behavior.

## fixes applied

### fix 1: added edge 6 for --retry

added to playtest artifact:

```markdown
### edge 6: retry failed workflows

**setup:** PR with failed CI checks (same as edge 2)

**action:**

\`\`\`sh
npx rhachet run --skill git.release -- --to main --mode apply --retry
\`\`\`

**expected:**

\`\`\`
🐢 lets try again...

🌊 release: turtle/your-branch → main
   ├─ 🔄 rerun triggered for 2 workflow(s)
   └─ 🥥 let's watch
\`\`\`

**pass criteria:** shows rerun confirmation, then watches
```
