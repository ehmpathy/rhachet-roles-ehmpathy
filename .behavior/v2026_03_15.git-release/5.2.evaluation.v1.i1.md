## summary (as implemented)

built a `git.release` skill that wraps the extant `git release` alias from `~/.bash_aliases`. the skill provides:

- `git.release --to main` вЂ” merge feature branch PR to main, watch CI
- `git.release --to prod` вЂ” full release cycle: merge в†’ watch test CI в†’ tag в†’ watch deploy/publish CI
- both with `--mode plan|apply` pattern (plan is default, apply executes)
- `--retry` flag to rerun failed workflows

the skill reuses:
- keyrack token fetch from git.commit.push
- plan/apply pattern from sedreplace
- turtle vibe output from output.sh
- tree output structure from git.commit skills
