## contracts

### input contract

```
cpsafe --from 'glob-pattern' --into 'dest-dir/'
mvsafe --from 'glob-pattern' --into 'dest-dir/'
rmsafe --path 'glob-pattern'
```

- glob pattern must be single-quoted to prevent shell expansion
- dest must be directory when glob matches >1 file

### output contract

```
🐢 sweet | heres the wave... | crickets...

🐚 {skill}
   ├─ from: {pattern}
   ├─ into: {dest}
   ├─ files: {count}
   └─ {verb}
      ├─ {file} -> {dest}
      └─ {file} -> {dest}
```

or for zero matches:

```
🐢 crickets...

🐚 {skill}
   ├─ from: {pattern}
   ├─ into: {dest}
   ├─ files: 0
   └─ {verb}
      └─ (none)
```
