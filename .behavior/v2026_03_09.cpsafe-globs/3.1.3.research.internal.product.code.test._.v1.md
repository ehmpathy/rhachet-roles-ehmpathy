## pattern.1: runInTempGitRepo helper

### [REUSE] temp git repo creation

**location**: cpsafe.integration.test.ts:17-55

```typescript
const runInTempGitRepo = (args: {
  files?: Record<string, string>;
  symlinks?: Record<string, string>;
  cpsafeArgs: string[];
}): { stdout: string; stderr: string; exitCode: number; tempDir: string } => {
  const tempDir = genTempDir({ slug: 'cpsafe-test', git: true });

  // create files
  if (args.files) {
    for (const [filePath, content] of Object.entries(args.files)) {
      const fullPath = path.join(tempDir, filePath);
      fs.mkdirSync(path.dirname(fullPath), { recursive: true });
      fs.writeFileSync(fullPath, content);
    }
  }

  // run cpsafe.sh
  const result = spawnSync('bash', [scriptPath, ...args.cpsafeArgs], {
    cwd: tempDir,
    encoding: 'utf-8',  // note: library API param name
    stdio: ['pipe', 'pipe', 'pipe'],
  });

  return {
    stdout: result.stdout ?? '',
    stderr: result.stderr ?? '',
    exitCode: result.status ?? 1,
    tempDir,
  };
};
```

**verdict**: reuse this pattern exactly. the `files` Record enables declarative test setup.

given('[caseN] glob patterns', () => {
  when('[t0] glob matches multiple files', () => {
    then('all files are copied', () => {
      const result = runInTempGitRepo({
        files: {
          'src/a.md': 'content a',
