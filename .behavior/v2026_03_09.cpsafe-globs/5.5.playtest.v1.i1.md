## happy paths

### case.1 = cpsafe glob copies all matches

```sh
# setup
mkdir -p src dest
echo "file a" > src/a.md
echo "file b" > src/b.md
echo "file c" > src/c.md

# execute
npx rhachet run --skill cpsafe --from 'src/*.md' --into 'dest/'

# verify
ls dest/
```

**expected output:**
```
🐢 sweet

🐚 cpsafe
   ├─ from: src/*.md
   ├─ into: dest/
   ├─ files: 3
   └─ copied
      ├─ src/a.md -> dest/a.md
      ├─ src/b.md -> dest/b.md
      └─ src/c.md -> dest/c.md
```

**pass criteria:**
- [ ] output shows turtle header "sweet"
- [ ] output shows files: 3
- [ ] all three files exist in dest/
- [ ] original files still exist in src/

### case.4 = recursive glob **/*.ts

```sh

rm -rf src archive
mkdir -p src/utils src/core archive
echo "util" > src/utils/foo.ts
echo "core" > src/core/bar.ts

# execute
npx rhachet run --skill cpsafe --from 'src/**/*.ts' --into 'archive/'

# verify
ls archive/
```

**expected output:**
```
🐢 sweet

🐚 cpsafe
   ├─ from: src/**/*.ts
   ├─ into: archive/
   ├─ files: 2
   └─ copied
      ├─ src/utils/foo.ts -> archive/foo.ts
      └─ src/core/bar.ts -> archive/bar.ts
```

**pass criteria:**
- [ ] output shows files: 2
- [ ] both .ts files copied to archive/
- [ ] original files intact in src/
