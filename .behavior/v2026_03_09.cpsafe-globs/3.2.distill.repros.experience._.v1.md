## reproduction feasibility

### all experiences: ✅ feasible

**test utilities available**:
- `genTempDir({ git: true })` - isolated git repo
- `spawnSync` - execute shell commands
- `fs` module - file assertions

**setup required**:
- create temp directory with git init
- write test files via `files: Record<string, string>`
- run skill via `spawnSync`

**concrete test sketch**:

```typescript
given('[caseN] batch copy with glob', () => {
  when('[t0] glob matches 3 files', () => {
    then('all files copied and output correct', () => {
      const result = runInTempGitRepo({
        files: {
          'src/a.md': 'a',
          'src/b.md': 'b',
          'src/c.md': 'c',
        },
        directories: ['dest'],
        cpsafeArgs: ['--from', 'src/*.md', '--into', 'dest/'],
      });

      // files copied
      expect(fs.existsSync(path.join(result.tempDir, 'dest/a.md'))).toBe(true);
      expect(fs.existsSync(path.join(result.tempDir, 'dest/b.md'))).toBe(true);
      expect(fs.existsSync(path.join(result.tempDir, 'dest/c.md'))).toBe(true);

      // output format
      expect(result.stdout).toContain('🐢 sweet');
      expect(result.stdout).toContain('🐚 cpsafe');
      expect(result.stdout).toContain('files: 3');
      expect(result.stdout).toContain('a.md -> dest/a.md');
    });
  });
});
```
