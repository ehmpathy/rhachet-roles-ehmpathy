# blackbox criteria: glob-aware safe fileops

## usecase.1 = cpsafe with glob patterns

```
given(a repo with multiple .md files in a directory)
  when(user runs cpsafe with glob pattern --from 'dir/*.md' --into 'dest/')
    then(all files that match the glob are copied to destination)
      sothat(user can batch copy without manual iteration)
    then(output shows turtle header "sweet")
    then(output shows each file as it is copied: source -> dest)
      sothat(user sees live progress)
    then(output shows total file count)

given(a repo with files in nested directories)
  when(user runs cpsafe with recursive glob --from 'src/**/*.ts' --into 'archive/')
    then(all files that match recursively are copied)
    then(output shows full relative paths: src/utils/foo.ts -> archive/foo.ts)

given(a glob pattern that matches zero files)
  when(user runs cpsafe --from '*.xyz' --into 'dest/')
    then(output shows turtle header "crickets...")
    then(output shows files: 0)
    then(output shows copied with (none) under it)
      sothat(user knows explicitly that no files matched)

given(a glob pattern that matches files outside repo)
  when(user runs cpsafe with pattern that would match external files)
    then(operation fails fast with security error)
      sothat(repo boundary is enforced)

given(glob matches multiple files but destination is a file not directory)
  when(user runs cpsafe --from '*.md' --into 'single.md')
    then(operation fails with clear error)
      sothat(user knows destination must be directory for multi-file ops)
```

## usecase.4 = sedreplace output phrases

```
given(a sedreplace command in plan mode)
  when(matches are found)
    then(output shows turtle header "heres the wave...")
      sothat(user knows this is a preview)

given(a sedreplace command in apply mode)
  when(replacements are made)
    then(output shows turtle header "sweet")
      sothat(user knows operation succeeded)

given(a sedreplace command)
  when(no matches are found)
    then(output shows turtle header "crickets...")
      sothat(user knows no files matched)
```

## usecase.5 = shell quoting for globs

```
given(documentation and permission examples)
  when(showing glob patterns)
    then(all examples use single quotes: --from '*.md')
      sothat(shell does not expand glob before skill receives it)
    then(no examples use double quotes or unquoted globs)
```

## exchange boundaries

### inputs
- `--from` / `--path`: glob pattern as string literal (single-quoted)
- `--into`: destination directory path
- `--mode`: plan | apply (for sedreplace)

### outputs
- turtle header: "sweet" | "heres the wave..." | "crickets..."
- shell header: skill name
- tree params: from, into, path, files count
- tree results: verb (copied/moved/removed) with file list printed incrementally

### errors
- glob matches files outside repo: fail-fast with security error
- destination is file when multiple sources: fail with clear error
- source does not exist: fail-fast
