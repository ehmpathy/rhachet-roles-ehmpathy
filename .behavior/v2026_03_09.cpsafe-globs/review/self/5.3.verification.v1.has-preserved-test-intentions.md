**why change was needed:** LLM outputs are probabilistic. ALL must pass is too strict for probabilistic verification. CI already used 'SOME'; local tests with LOCALLY=true should match.

**intention preserved?** yes - test still verifies "LLM can detect prompt injection". the change adjusts retry tolerance, not the detection assertion. detection must still succeed on at least 1 of 3 attempts.
