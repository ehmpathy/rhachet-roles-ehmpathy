# research: production code patterns

## summary

researched extant production code to identify patterns for implementation of git.stage.add + unlimited quota features.

note: after rebase, architecture changed:
- `git.commit.uses.sh` is now a dispatcher (routes to local.sh or global.sh)
- `git.commit.uses.local.sh` handles per-repo quota
- `git.commit.uses.global.sh` handles global blocker
- `allow` command uses infinite for unlimited (not infinite)
