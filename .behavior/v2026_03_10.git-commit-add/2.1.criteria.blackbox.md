# blackbox criteria: git.stage.add + unlimited quota

## usecase.1 = grant stage permission

```
given('human wants mechanic to stage files')
  when('human runs git.commit.uses set --quant 5 --push allow --stage allow')
    then('quota file is created with stage: allow')
      sothat('mechanic can use git.stage.add')
    then('output shows granted permissions')

  when('human runs git.commit.uses set --quant 5 --push allow')
    then('quota file is created with stage: block (default)')
      sothat('backwards compatibility is preserved')
```

## usecase.2 = grant unlimited quota

```
given('human wants to give mechanic unlimited commits')
  when('human runs git.commit.uses allow')
    then('quota file shows uses: infinite, push: allow, stage: allow')
    then('output shows unlimited granted')

  when('mechanic commits multiple times')
    then('uses is not decremented when uses === "infinite"')
      sothat('mechanic can work without quota anxiety')
```

## usecase.3 = revoke permissions

```
given('human wants to end a session')
  when('human runs git.commit.uses del')
    then('quota file is removed entirely')
    then('output confirms revocation')
      sothat('session ends cleanly')

  when('human runs git.commit.uses del with no quota file')
    then('command succeeds silently')
      sothat('del is idempotent like rm -f')
```

## usecase.4 = stage specific files

```
given('mechanic has stage permission')
  when('mechanic runs git.stage.add src/file1.ts src/file2.ts')
    then('files are staged for commit')
    then('output shows tree-formatted list of staged files')
    then('uses quota is not decremented')
      sothat('stage is preparatory, not countable')

  when('mechanic runs git.stage.add with file not found')
    then('error: file not found: path/to/file.ts')
    then('no files are staged')
      sothat('fail-fast prevents partial state')
```

## usecase.5 = stage all with dot

```
given('mechanic has stage permission')
  when('mechanic runs git.stage.add .')
    then('all files in current directory are staged')
    then('output shows staged: .')
      sothat('mechanic can stage all files in one command')
```

## usecase.6 = stage without permission

```
given('mechanic has commit quota but no stage permission')
  when('mechanic runs git.stage.add src/file.ts')
    then('error: stage not allowed')
      sothat('permission model is enforced')
```

## usecase.7 = idempotent stage

```
given('mechanic has stage permission')
  when('mechanic stages a file that is already staged')
    then('operation succeeds (no-op)')
      sothat('re-stage is idempotent')
```

## usecase.8 = path validation

```
given('mechanic has stage permission')
  when('mechanic runs git.stage.add /etc/passwd')
    then('error: path must be within repo')
      sothat('files outside repo cannot be staged')
```

## usecase.9 = check quota

```
given('quota was set with allow command')
  when('mechanic runs git.commit.uses get')
    then('output shows uses: infinite')
    then('output shows push: allowed')
    then('output shows stage: allowed')
      sothat('mechanic can see all permissions')
```

## usecase.10 = typical workflow

```
given('human starts a work session')
  when('human grants: git.commit.uses allow')
    then('mechanic can stage, commit, and push freely')

  when('mechanic completes work')
    then('mechanic stages files: git.stage.add src/feature.ts')
    then('mechanic commits: git.commit.set -m @stdin --mode apply')
    then('mechanic pushes: git.commit.push --mode apply')

  when('human ends session: git.commit.uses del')
    then('all permissions revoked')
    then('clean slate for next session')
```

## usecase.11 = global blocker check

```
given('global blocker is active')
  when('mechanic runs git.stage.add src/file.ts')
    then('error: globally blocked')
      sothat('global blocker overrides local permissions')
```
