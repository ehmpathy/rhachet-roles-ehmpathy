# distill: user experience reproductions

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| grant stage permission | git.commit.uses set | --quant N --stage allow | quota file shows stage: allow | integration |
| grant unlimited quota | git.commit.uses set | --quant infinite --push allow | quota file shows uses: "infinite" | integration |
| revoke permissions | git.commit.uses del | (no args) | quota file removed | integration |
| stage specific files | git.stage.add | src/file1.ts src/file2.ts | files staged | integration |
| stage via bounded glob | git.stage.add | --glob 'src/**/*.ts' | matched files staged | integration |
| block unbound glob | git.stage.add | --glob '**/*.ts' | error: unbound glob | integration |
| check quota | git.commit.uses get | (no args) | shows uses, push, stage | integration |
| commit with infinite | git.commit.set | -m @stdin --mode apply | uses not decremented | integration |
| stage without permission | git.stage.add | src/file.ts | error: stage not allowed | integration |
| idempotent del | git.commit.uses del | (no quota file) | succeeds silently | integration |

### setup required

```ts
const runInTempGitRepo = (args: {
  args: string[];
  files?: Record<string, string>;
  meterState?: { uses: number | string; push: string; stage?: string };
}) => {
  const tempDir = genTempDir({ slug: 'git-stage-test', git: true });

  // create files
  if (args.files) {
    for (const [filePath, content] of Object.entries(args.files)) {
      const fullPath = path.join(tempDir, filePath);
      fs.mkdirSync(path.dirname(fullPath), { recursive: true });
      fs.writeFileSync(fullPath, content);
    }
    execSync('git add .', { cwd: tempDir });
    execSync('git commit -m "initial"', { cwd: tempDir });
  }

  // create meter state
  if (args.meterState) {
    const meterDir = path.join(tempDir, '.meter');
    fs.mkdirSync(meterDir, { recursive: true });
    fs.writeFileSync(
      path.join(meterDir, 'git.commit.uses.jsonc'),
      JSON.stringify(args.meterState, null, 2),
    );
  }

  const result = spawnSync('bash', [skillPath, ...args.args], {
    cwd: tempDir,
    encoding: 'utf-8',
    stdio: ['pipe', 'pipe', 'pipe'],
  });

  return { stdout: result.stdout, stderr: result.stderr, exitCode: result.status, tempDir };
};
```
