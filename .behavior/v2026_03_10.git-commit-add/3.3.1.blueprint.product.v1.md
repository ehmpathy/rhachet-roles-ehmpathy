## test coverage

### git.commit.uses.integration.test.ts [~]

| case | test |
|------|------|
| infinite quota | set --quant infinite → state shows "infinite" |
| infinite get | get with infinite → displays "infinite" |
| --stage allow | set --stage allow → state shows stage: allow |
| --stage block | set --stage block → state shows stage: block |
| --stage default | set without --stage → state shows stage: block |
| del | del → state file removed |
| del idempotent | del with no file → succeeds silently |
| get with stage | get → displays stage permission |

### git.commit.set.integration.test.ts [~]

| case | test |
|------|------|
| infinite no decrement | commit with infinite → uses stays "infinite" |

### git.stage.add.integration.test.ts [+]

| case | test |
|------|------|
| stage files | stage src/file.ts → file staged |
| no permission | stage without --stage allow → error |
| file not found | stage absent.ts → error |
| bounded glob | --glob 'src/**/*.ts' → files staged |
| unbound glob | --glob '**/*.ts' → blocked |
| already staged | stage staged file → no-op |
| path outside repo | stage /etc/passwd → error |
| no files | stage with no args → error |
| glob no match | --glob '*.xyz' → error |
| output format | stage → tree format output |
