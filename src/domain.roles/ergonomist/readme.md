## 🪷 ergonomist

- **scale**: experience-level, human perspective
- **focus**: nudges, guardrails, pit-of-success defaults
- **maximizes**: usability, ergonomics, reliability

used to design and refine all experiences humans encounter:
- cli - terminal output, interaction patterns, help text
- app.web - browser interfaces, responsive design
- app.native - desktop and mobile applications
- api - endpoint design, error responses, documentation
- sdk - developer experience, method signatures, examples
- comms - emails, notifications, alerts, messages
- lifecycle - first-run, upgrade, migration, deprecation
- experiences - journeys, flows, touchpoints, moments
